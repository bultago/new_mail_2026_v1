개인정보 설정

개인 정보를 수정합니다.

  1. [환경설정 > 개인정보] 메뉴를 클릭합니다.
  2. 개인정보화면에서 로그인한 계정의 비밀번호를 입력합니다.
  3. 인증하기 버튼을 클릭합니다.
  4. 개인 정보 수정 화면으로 전환되고, 개인 정보 화면에서 정보를 수정합니다.
    • 기본정보 – 개인 사진을 업로드하고 기본 정보를 입력합니다.
    • 개인정보 – 개인 정보를 입력합니다.
    • 회사정보 – 개인의 회사 정보를 입력합니다.
  5. 수정이 완료되면 저장하기 버튼을 클릭합니다.
  6. 수정 완료 메시지 창이 나타납니다. 확인버튼을 클릭합니다.
