각 메일함에 있는 메일을 검색할 수 있습니다.

  • 빠른 검색 – 깃발메일, 안읽은 메일 등 구분된 메일을 필터링하여 검색합니다.
  • 기본 검색 - 보낸이/받는이, 제목으로 메일을 검색합니다.
  • 상세 검색 - 보낸이/받는이, 검색 기간 등 검색 조건을 설정하여 검색합니다.

 
빠른 검색

빠른검색 기능을 통해 현재 선택한 메일함의 메일 목록에서 깃발메일, 안읽은 메일 등을 필터링하여 확인할 수 있습니다.

  1. 메일 목록에서 기본기능탭의 빠른검색을 클릭합니다.
  2. 다음과 같은 보기조건 목록이 나타납니다.
    • 깃발메일, 안읽은메일, 읽은메일, 내게쓴메일, 답장한메일, 첨부메일, 오늘온메일, 어제온메일
  3. 보기조건 항목을 클릭하면 해당하는 메일이 필터링되어 나타납니다.

 
기본 검색

기본 검색은 보낸이/받는이, 제목의 검색어를 입력하여 검색하는 기능입니다.

  1. [메일] 메뉴 화면 우측 상단에서 검색란에 검색하려는 검색어를 입력합니다.
  2. 검색버튼을 클릭하면, 검색결과 목록이 나타납니다.

[참고]
검색 확인 후, 검색입력란에서 를 클릭하면 검색하기 전단계화면으로 이동합니다.

 
상세 검색

상세 검색은 보낸이/받는이, 검색 기간 등 검색 조건을 상세하게 설정하여 검색하는 기능입니다.

  1. [메일] 메뉴 화면 우측 상단에서 상세검색을 클릭합니다.
  2. 다음과 같은 상세검색 조건창이 나타납니다. 각 검색 조건을 입력합니다.

    항 목 설 명
    보낸 사람 보낸 사람의 계정을 입력합니다.
    받는 사람 받는 사람의 계정을 입력합니다. 참조를 포함하여 검색합니다.
    검색어

    검색 범위를 선택한 후 검색어를 입력합니다.

    메일함

    검색할 메일함을 선택합니다.

    [참고]
    [환경설정 > 기본환경]의 검색 범위 설정에서 스팸메일함, 휴지통 포함을 체크한 경우, 상세검색시 메일함을 전체메일함으로 선택하면 스팸메일함과 휴지통에 있는 메일이 포함되어 검색됩니다.

    [참고]
    웹 어드민의 [환경설정 > 기타 > 검색 엔진 설정]에서 검색 범위(메일 본문 검색, 첨부파일 내용 검색)를 설정할 수 있습니다.
    검색기간 텍스트 박스를 클릭하면 달력이 나타납니다. 달력에서 날짜를 선택하여 검색기간을 설정합니다.
    부가 조건 깃발메일, 안읽은 메일 등으로 메일을 필터링하여 검색합니다.

  3. 검색 조건 설정이 완료되면, 검색버튼을 클릭합니다.

[참고]
검색 확인 후, 검색입력란에서 를 클릭하면 검색하기 전단계화면으로 이동합니다.
