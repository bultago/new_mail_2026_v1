외부메일 설정은 다른 메일서비스에 따로 접속하지 않고, 외부 메일 계정에 수신된 메일을 확인할 수 있는 기능입니다. 단, 해당 메일 서비스는 POP과 POPS 서버를 지원하는 경우에만 외부메일을 가져올 수 있습니다. 외부메일 설정은 최대 7개까지 설정할 수 있습니다.

 
외부메일 관리

외부메일 서버 정보를 추가, 수정, 삭제하여 관리합니다.

외부 메일 추가

  1. [환경설정 > 외부메일설정] 메뉴를 클릭합니다.
  2. 외부메일 추가에서 외부메일 정보를 입력합니다.

    항 목 설 명
    POP3 서버명 외부메일의 서버 Host 명 또는 IP를 입력합니다.
    또는 우측 서버 목록에서 메일서버를 선택합니다. 메일서버 목록에서 서버를 선택하면 POP3 서버명과 연결포트가 자동으로 입력됩니다. 해당 외부메일 서버에서 POP과 POPS를 모두 지원하면 POPS(995)가 기본으로 설정됩니다.
    연결포트 POP3 포트를 입력합니다. 기본값은 110입니다.
    보안연결(SSL)을 체크하면 POP에 SSL을 적용한 POPS 프로토콜을 사용합니다. POPS의 기본 포트는 995입니다
    아이디 해당 외부메일 로그인 계정 아이디를 입력합니다.
    비밀번호 해당 외부메일 로그인 계정의 비밀번호를 입력합니다.
    원본삭제 여부 외부메일 서버에서 메일을 가져온 후, 원본 메일을 삭제할지에 대한 여부를 선택합니다.
    저장 메일함 외부메일 서버에서 메일을 가져왔을 때 저장하는 메일함을 지정할 수 있습니다.
    새 메일함을 추가할 경우, 상위 메일함을 선택하여 새 메일함 명을 입력합니다.

  3. 외부메일 정보 입력이 완료되면 저장하기버튼을 클릭합니다.
  4. 저장 완료 메시지 창이 나타납니다. 확인버튼을 클릭합니다.

 

외부 메일 수정

외부 메일 목록에 있는 외부메일 정보를 수정합니다.

  1. [환경설정 > 외부메일설정] 메뉴를 클릭합니다.
  2. 외부 메일 목록에서 수정할 외부메일 체크한 후, 수정 버튼을 클릭합니다.
  3. 외부 메일 추가에 설정 정보가 나타납니다. 외부 메일 정보를 수정합니다.
  4. 수정하기 버튼을 클릭합니다.
  5. 수정 완료 메시지 창이 나타납니다. 확인버튼을 클릭합니다.

 

외부 메일 삭제

외부 메일 목록에 있는 외부메일을 삭제합니다.

  1. [환경설정 > 외부메일설정] 메뉴를 클릭합니다.
  2. 외부 메일 목록에서 삭제할 외부메일을 선택한 후, 삭제버튼을 클릭합니다.
  3. 삭제 확인 메시지 창이 나타납니다. 확인버튼을 클릭합니다.
  4. 삭제 완료 메시지 창이 나타납니다. 확인버튼을 클릭합니다.

 

 
외부메일 가져오기

설정한 외부메일 가져오기는 [메일]메뉴에서 수행합니다.
