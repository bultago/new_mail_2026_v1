게시판에 등록된 게시물을 검색합니다.

 
전체 검색

전체 게시판에서 등록된 게시물을 검색합니다.

  1. [게시판]메뉴의 화면 우측 상단에서 검색 조건을 선택합니다.
    • 제목, 내용, 작성자, 첨부파일명, 제목+내용
  2. 검색어를 입력합니다.
  3. 검색버튼을 클릭합니다.

 
내가 작성한 게시물 검색

전체 게시판에서 내가 작성한 게시물만 검색합니다.

  1. [게시판]메뉴의 화면 우측 상단에서 내가쓴글을 클릭합니다.
  2. 내가 작성한 게시물의 검색결과가 나타납니다.
