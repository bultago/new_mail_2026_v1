1. Read Messages

You can view inbound and outbound messages in each folder(mailbox).

  • Select a folder - If you click Read, you'll go to Inbox. If you choose another folder, you'll move to it. If you select Etc, you will go to Select a folder displaying all the folders including personal folders and can choose one.

  • Read/Unread - Messages can be classified according to Read/Unread status.

  • Move - After selecting messages you'd like to move, click Move and select a folder.

  • View message contents - You can view message contents by clicking a subject of a message.

  • Reply - You can reply to a message by clicking Reply. If you reply, To will be filled with only the sender's email address and Re: will be written before subject automatically.

  • Reply to all - You can reply to a message by clicking Reply to all. To will be filled with the sender, cc and bcc's email address.

  • Forward - You can forward a message you received to other accounts. If you forward a message, [Fwd] will be written before subject automatically.

2. Compose Message

You can send a message after entering recipients and a message body.

  • Designate Recipient - Designate recipients into To, Cc or Bcc when sending a message.

  • Subject - Enter subject.

  • Write a message body - When you write a message body, you can write up to 10,000 bytes (20KB). Text editor is only supported on mobile devices.

  • Save - You can save a message in the middle of composing one by clicking Save.

3. Help

Online help describes how to use menus when you connect Terrace Mail Suite via mobile devices.

4. Sign out

You can terminate a login session.
