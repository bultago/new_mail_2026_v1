Filter Management

 

Filters allows you to manage the flow of incoming messages. Using filters, you can automatically move messages into a certain mailbox.
You can use items for filters as below.

  • Filter Items - Sender/Recipient email addresses, subject

OR condition is applied to each item.

 
Add a Filter

To add a filter to your messages, follow the steps below.

  1. Go to [Mail] and check a message that you'd like to add a filter.

    2. [NOTE]
    You can only add one filter at one time in [Mail].

  2. Click Filter on the More tab menu.
  3. Created Filters window appears with items filled with information(sender, recipient, subject) of the message.
    If you'd like to change items, edit items on Created Filters.
    • Sender Address(From) - Incoming messages with the same sender will be automatically moved.
    • Receiver Address(To/Cc) - Outgoing messages with the same reciever or cc will be automatically moved.
    • Subject - Messages containing the same subject will be automatically moved.
    • Above messages will be moved to - Designate a folder where messages will be moved into.
    • Create a new folder - If you'd like to create a new folder, check Create a new folder and select parent folder.
  4. Click Submit. If you'd like to undo, click Close.

 
Filter Management

You can add, edit and delete filters.

[NOTE] Refer to [Options > Filters] to learn more about how to manage filters.
