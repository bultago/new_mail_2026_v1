You can manage your data by uploading or downloading into your web folder. You can give authority to other users so that they can view or edit your data.

 
Folder Management

You can manage your data by grouping them by folders.
To add, edit, delete folders that you use to manager your data, follow the steps below.

Add Folder

You can add folders to My Web Folder up to 4 depths.
To add folders, follow the steps below.

  1. Click a folder that you'd like to create in [Web Folders].
    If you haven't created a folder, the only folder existed is My Web Folder which is provided by default.
  2. Click New on the Basic tab menu.
  3. New window will be popped up.
  4. Enter a new folder name and click Submit.
    • Folder name should be between 2 and 32 bytes long.

Modify Folder

To rename folders of My Web Folder, follow the steps below.

  1. Check a folder that you'd like to edit in [Web Folders].
  2. Click Rename on the Basic tab menu.
  3. Enter a new folder name into New Name when Rename window pops up and click Submit.
  4. A completion message will be prompted. Click OK.

Delete Folder

To delete folders from My Web Folder, follow the steps below.

  1. Check folders that you'd like to delete in [Web Folders].
  2. Click Delete on the Basic tab menu.
  3. A confirmation message will be prompted. Click OK.
  4. A completion message will be prompted. Click OK.

 

You should note below when you delete a folder.

  • When a folder that you try to delete has sub folders - Delete sub folders first and then delete the folder.
  • When a folder that you try to delete is shared - Cancel sharing first and then delete the folder.
  • When a folder that you try to delete has files in it - All the files in the folder will be deleted.

 
Move/Copy Folder

To move files or folders, follow the steps below.

Move File/Folder

To move files or folders, follow the steps below.

  1. Check files or folders that you'd like to move in [Web Folders > My Web Folder].
  2. Click Move on the Basic tab menu.
  3. When Select folder window appears, select a folder.
  4. A completion message will be prompted. Click OK.

 

Copy File/Folder

  1. Check files or folders that you'd like to copy in [Web Folders > My Web Folder].
  2. Click Copy on the Basic tab menu.
  3. When Select folder window appears, select a folder.
  4. A completion message will be prompted. Click OK.

 
Send an Email

To send a message attached with files that you uploaded into My Web Folder, follow the steps below.

  1. Check files that you'd like to send in [Web Folders > My Web Folder].
  2. Click Send message on the Basic tab menu.
  3. When Compose window appears with the file attached, write an email.
  4. Click Send after composing a message.

[NOTE]
Attachment methods (general attachments, large attachments) cannot be changed because files in Web Folder are already uploaded.

 
File Upload/Download

You can upload and download files.

Upload File

  1. Select a folder that you'd like to upload to in [Web Folders > My Web Folder].
  2. Click Attach Files next to File Upload.
    • Upload - Files are attached by clicking Attach Files button.
    • Use ActiveX - Files are attached by using Drag&Drop.
  3. Select files that you'd like to upload from your PC and click Open.
  4. Click Upload.

[NOTE]
If you upload files with the same name, system recognizes those as different files.

[NOTE]
Virus scan for upload files carry out when the file is not large and Activation of Virus Scan in [Filters > Virus] is enabled. As long as the size of upload file is below Max File Size to Scan in [Filters > Virus], virus scan can be done.

Download File

  1. Select a folder that you'd like to download in [Web Folders > My Web Folder].
  2. Click a file that you'd like to download.

    3. [NOTE]
    If you want to download severl files together, check files you'd like to download and click File Download on the Basic tab menu.

  3. Click Open or Save to download it to your PC.

Delete File

  1. Check files that you'd like to delete in [Web Folders > My Web Folder].
  2. Click Delete on the Basic tab menu.
  3. A confirmation message will be prompted. Click OK.
  4. A completion message will be prompted. Click OK.

 

 
Shared Folder Management

You can read or write folders that other uses share with you. You also can share your folders with other users.

Folder Share Configuration

To share your folders with other users, follow the steps below.

  1. Click [Web Folders > My Web Folder].
  2. Select a folder that you'd like to share and click Share on the More tab menu.
  3. When Share window appears, enter items below.

    Item Description
    Folder Name It shows the folder name.
    Description Enter a description or purpose of the shared folder.
    Total Shared

    If you check Total Shared, the folder you've checked will be shared with all the users. Select either Read Auth or Read/Write Auth. If you select Read Auth, the shared folder will only be viewed by other users. If you select Read/Write Auth, the shared folder will be viewed and written by other users.
    Specification User Share

    You can select some users specifically who you will share with.

    1. Search for a user or an organization.
      • Select ID or Name and enter one of them.
      • Select Org and choose an organization.
        If you check sub-org, the folder will be shared with the organization including sub organization.
    2. Select users that you'd like to share with and click Add.

    4.  
  4. Click OK when you're done.
  5. A completion message will be prompted. Click OK.
  6. Once you share folders, they will appear with Shared Folder Icon().

Shared Folder Information Modification

To change sharing information, follow the steps below.

  1. Click [Web Folder > My Web Folder].
  2. Select one of the shared folders that you'd like to change.
  3. Click Sharing Info on the More tab menu.
  4. Share window will be popped up. Enter items.
  5. Click OK to save your settings.
  6. A completion message will be prompted. Click OK.

Cancel Folder Sharing

To cancel sharing your folders, follow the steps below.

  1. Click [Web Folder > My Web Folder].
  2. Select one of the shared folders that you'd like no longer to share.
  3. Click Sharing Off on the More tab menu.
  4. A confirmation message will be prompted. Click OK.
  5. A completion message will be prompted. Click OK.
  6. Once the folder is no longer shared, the folder will appear as the ordinary folder icon().
