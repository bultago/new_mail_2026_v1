Read Notes

 

Note is a memo that can be sent to or received from webmail users in the same domain.

[NOTE]
The function, Note, is not offered by default. You have to buy the license if you'd like to use this function.

 
Read Notes

You can see sender(recipient)'s email address, date and contents of notes.

There are Inbox, Sent, Stored.

Inbox

You can read received notes in Inbox. Inbox consists of items below.

Item Description
Checkbox You can check notes when you'd like to delete, save or reply to notes.
Status It shows Read/Unread(/) status of received notes.
From It shows sender's name or email address.
Contents It shows contents of a note. If you click the contents, Read window will be popped up.
Date It shows the date you received a note.
 

Sent

You can see notes you sent. Sent consists of items below.

Item Description
Checkbox You can check notes when you'd like to delete, save or reply to notes.
Status It shows status of sent notes. It is always Read() status.
Recipient It shows recipient's name or email address.
Contents It shows contents of a note. If you click the contents, Read window will be popped up.
Date It shows the date you sent a note.
Receipt You can see if recipient has read the note or not.
It shows the date and time that recipients see the note if the note has been read.
Recall button appears if the note has not been read. You can recall the note by clicking Recall.
 

Stored

You can see notes that you saved. Stored consists of items below.

Item Description
Checkbox You can check notes when you'd like to delete, save or reply to notes.
Status It shows Read/Unread(/) status of received/sent notes.
Sender/
Recipient		 It shows sender or recipient's name or email address.
Contents It shows contents of a note. If you click the contents, Read window will be popped up.
Date It shows the date you received or sent a note.
 

[NOTE]
Notes in Inbox and Sent will be automatically deleted after Valid period for notes of [Services > Account > Group Management] and [Services > Account > Users] in admin page. If you'd like to save notes for a long time, save them in Stored.
