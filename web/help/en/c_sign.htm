Change Signature

 

You can add and change your signature. Once you set your signature, that will be automatically inserted to messages you will write.

  1. Click [Options > Signature].
  2. Select either Use or Not use in Default Signature.
  3. Select Position of the sign for the transfer.
  4. Click Save once you finish setting.

 
Signature Settings

Select Attach Signature and Position of the signature for the transfer.

Item Description
Attach Signature

Designate whether to insert or not your signature when you write a message.

  • Attach, Not Attach
Position of the sign for the transfer

Designate where your signature will be displayed when you forward or reply to a message.

  • Bottom of all message - Your signature will be attached below all the messages included.
  • Bottom of the message being created - Your signature will be attached right below the message you're createing.

Add Signature

To add a new signature, follow the steps below.

  1. Click [Options > Signature].
  2. Click Add on Signature Edit.
  3. Set your signature when Signature Edit window prompts.
    • Signature Subject - Enter a signature title.
    • Default Signature - Set the signature as a default one. When multiple signatures exist, Default Signature will be attached by default.
    • Upload Signature image - Upload an image as your signature.
    • Signature Editor - Select editor (HTML, TEXT). Or, you can select a signature template (Template1, Template2, Template3) and add your information easily.
  4. Check an added image and enter text to be used in your signature.
  5. Click Save once you finish setting.
  6. A completion message will be prompted. Click OK.

[NOTE]
If you use a text editor, a signature template might not be displayed properly.

Modify Signature

To modify your signature, follow the steps below.

  1. Click [Options > Signature].
  2. Select a signature in Signature Edit and click Modify.
  3. Change items you'd like to modify once Edit Signature window will appear.
  4. Click Save.

Delete Signature

To delete your signature, follow the steps below.

  1. Click [Options > Signature].
  2. Select a signature in Signature Edit and click Delete.
