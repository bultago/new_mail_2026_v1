Import Messages

 

Import Mail can download messages from up to seven other email accounts. It helps you centralize all your messages in TMS. Note that importing messages can be as long as the mail server supports POP and POPS.



Import Mail Account Settings

You can add, modify and delete your email accounts.

Add Mail Accounts

  1. Click [Options > Import Mail].
  2. Enter your mail information in Add Mail Account.

    Item Description
    POP Server Enter a host name or IP address of the mail(POP) server you'd like to add. Or, you can select one from POP Server. If you select a mail server from the list, a server name and port will be entered automatically. If the mail server you have selected support both POP and POPS, POPS(995) will be set as default.
    Port Enter port number. The default number of POP is 110.
    If you check Secure Connection with SSL, POPS protocol which adds SSL over POP will be used. The default port number for POPS is 995.
    Account Enter your account.
    Password Enter your password.
    Leave/Delete Select what action will be done after messages are imported. Select either keep the messages or delete them from the server.
    Folder Select a mailbox which will store messages from the account you've entered above. If you create a new mailbox, select parent's mailbox and then type name of a new mailbox into a text box.

  3. Click Save when finished.
  4. A completion message will be prompted. Click OK.

 

Modify Mail Accounts

You can edit your mail accounts added to import messages.

  1. Click [Options > Import Mail].
  2. Click Modify after checking a mail account you'd like to edit from Added Mail Accounts.
  3. Your mail account information will be displayed in Add Mail Account. Edit it.
  4. Click Modify.
  5. A completion message will be prompted. Click OK.

 

Delete Mail Account

You can delete your mail accounts added to import messages.

  1. Click [Options > Import Mail].
  2. Click Delete after checking a mail account you'd like to delete from Added Mail Accounts.
  3. A completion message will be prompted. Click OK.

 

 
Import Messages
Messages can be imported by clicking [Mail > Import Mail].

[NOTE]
You can refer to [Mail> Read Messages> Import Mail] to learn how to import messages.
