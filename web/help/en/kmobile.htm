1. Read Messages

You can view inbound and outbound messages in each folder(mailbox).
If you sign in, you will be moved to Inbox by default.
  • Folder List - If you click Folder List, mailbox list including personal folders will appear. If you click a mailbox name, you will go to the mailbox.

  • Select All - If you click Select All, all the messages in the page will be selected. You can select all messages and delete, move or mark as read/unread.

  • Read/Unread - Messages can be classified according to Read/Unread status.

  • Reply - You can reply to a message by clicking Reply. If you reply, To will be filled with only the sender's email address and Re: will be written before subject automatically.

  • Reply to all - You can reply to a message by clicking Reply to all. To will be filled with the sender, cc and bcc's email address.

  • Forward - You can forward a message you received to other accounts. If you forward a message, [Fwd] will be written before subject automatically.

  • Delete - You can delete messages and those will go to Trash.

  • Move - You can move messages to Inbox, Spam, Trash or personal folders.

2. Compose Messages

You can send a message after entering recipients and a message body.

  • To - Designate recipients into To. This is a mandatory field.

  • Cc/Bcc - Designate recipients who get the message with To. Once you click (+) next to Cc/Bcc, Cc/Bcc fields will appear.

  • To Myself - If you'd like to send a message to yourself, click To Myself.

  • Subject - Enter a subject of the message.

  • Body - Write a message body and set Read Status and Signature.
    • Read Status - You can confirm whether the recipients read the message once you check Read Status.
    • Signature - If you'd like to add your signature to the body of the message, click Signature and select one. You can create and register your signature by connecting webmail. You are only allowed to choose a signature as long as you connect TMS via mobile device.
    • Contacts - You can send a message to a member or a group saved in Contacts.
    • Draft - Save a message you're writing into Draft.
