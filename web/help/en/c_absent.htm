Vacation Responder
Vacation Responder Settings

 

Vacation Responder let people know you won't be able to get back to them right away by sending a message automatically when you received one.

 
Vacation Responder Settings

To set Vacation Responder, follow the steps below.

  1. Click [Options > Vacation Responder].
  2. Set items below.
    • Response Mode Use - Select Apply or Not Apply.
    • Vacation Period - Specify how long you are away.
    • Subject - Enter summary of the message.
    • Message - Enter the body of your message.
  3. Select the recipient of auto-response message.
    • All the incoming messages - Send a response to all people who email you.
    • Only the messages in the auto response address list - Send a response to only those who in Response List.
      Response List will only be showed when Apply is selected in Response Mode Use.

      To manage Respond List, refer below.
      Functions Description
      Add mail addresses
      1. Type a mail address into a text box in Response List.
        (eg)daou_user@daou.co.kr
      2. Click Add.
      3. The address will be added.
      Delete mail addresses
      1. Select a mail address you'd like to delete from Response List.
      2. Click Delete.
      Search mail addresses
      1. Type text into a text box in Response List.
      2. Click Search.

  4. Click Save when finished.
  5. A completion message will be prompted. Click OK.
