To set preferences, follow the steps below.

  1. Click [Options > General].
  2. You can set Preferences so that you can read or write a message efficiently.
    • Preferences
    • Preferences on Reading Messages
    • Preferences on Composing Messages
  3. Click Save button once you finish setting.
    Click Default to initialize Preferences.

 
Preferences

Items of message preferences are as follows.

Item Description
Display Language

Language of TMS web page.

  • Korean, English, Japanese
The count of per page

How many messages will be displayed in a page.

  • 15, 20, 25, 30, 50, 80
New message time interval

You will be informed of a new message with the time interval.

  • Not Use, 1min, 5min, 10min, 30min, 1hour
Web interface

Select an interface of Terrace Mail Suite.

  • HTML
  • AJAX - It is faster than HTML because data download without refreshing. However, some browsers might not support it.
Search scope set

If this is enabled, Spam and Trash folder will be included when you search.
Upload ActiveX

You can set whether to use ActiveX by default when you compose a new message. Even though you set this by default here, you can change it whenever you compose a new message.

  • Enable, Disable

[NOTE]
This menu is shown only when ActiveX usage when uploading attachments is set as Enable in [Setting > Etc > Web Option Settings] in admin page.

[NOTE]
ActivceX is only compatible with Internet Explorers.

 

 
Preferences on Reading Messages

Items of preferences on reading messages are as follows.

Item Description
View images

If there are images in a message, you can set whether they will be showed by default.

[NOTE]
This option will be effect only when an image is inserted instead of being attached.
Disable HTML Tags

Inactive HTML tags which are harmful aiming at phishing or advertising.

 

 
Preferences on Composing Messages

Items of preferences on composing messages are as follows.

Item Description
Save sent message

Set whether to save messages you send.

  • Save, Not Save
Return Receipt

Set whether to require receipt of messages you send.

  • Return Receipt, No Receipt
Editor Mode

Set editor which will use when composing messages.

  • HTML, TEXT
Compose Mode

Designate which way you will compose a message.

  • Normal, Pop-up
Alert on draft messages

Set whether you will be informed if there are messages in Drafts.

  • Enable, Disable
Character sets

Designate which character set you will use.

  • Korean(EUC-KR), English(US-ASCII), Japanese(ISO-2022-JP), Chinese(GB2312), Chinese(BIG5), Unicode(UTF-8)
Nickname

Text your nickname. This nickname will be displayed as a sender to recipients. (eg) Human Resources

Reply address

If you'd like to receive a reply message to another mail address, enter your account. (eg) user@daou.co.kr

Attach V-card

Set whether to attach V-card in messages.

  • V-Card Management: Move to V-Card Management in Options.
