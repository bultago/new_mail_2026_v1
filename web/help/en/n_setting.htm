Notes Settings

You can set Block policy and manage blocking and allowed addresses.

  1. Click [Note > Settings].
  2. Set Block policy.
    • Allow all - Allow all the notes.
    • Reject all – Reject all the notes.
    • Allow notes from allowed users - Allow notes if you receive them from allowed addresses.
    • Reject notes from blocking users - Reject notes if you receive them from blocking addresses.
  3. If you set Block policy as Allow notes from allowed users or Reject notes from blocking users, add addresses that will be blocked or allowed.
  4. Click Save once you finish setting it.
