You can search for your messages in your mailboxes.

  • Quick Search - Search by flagged, unread messages.
  • Standard Search - Search by entering queries of subject, sender or recipient.
  • Advanced search - Search by entering queries and date range.

 
Quick Search

You can view your messages in a mailbox you've selected by criteria such as flagged, unread, read at once.

  1. Go to [Mail] and click a mailbox that you'd like to search. Click Quick Search on the Basic tab menu.
  2. Search criteria will appear as below.
    • Flagged, Unread, Read, Messages to, Reply, Attached, Today, Yesterday
  3. If you select a condition, messages which are matched with the condition will be displayed.

 

 
Standard Search

You can search by entering queries.

  1. Go to [Mail] and enter a search keyword on right top of the screen.
  2. Click Search and then you can find search results.

[NOTE]
If you click a button() next to Search, window will go to the previous page.

 

 
Advanced Search

Advanced Search enables you to search messages more specifically by entering detail information like mailbox, date range, keywords of subject, sender or recipient.
To use Advanced Search, follow the steps below.

  1. Go to [Mail] and click Advanced Search on right top of the screen.
  2. Once Advanced Search window will be popped up, enter items below.
    3. Item Description
    From Enter a keyword for the sender.
    To (Include TO/CC) Enter a keyword for the recipient.
    Query

    Select search criteria and enter a keyword.
    Folder Select a mailbox for search.

    [NOTE]
    If you check Include messages in Spam and Trash in [Options > General], messages in Spam and Trash will be included when searching.

    [NOTE]
    Administrator can set a range of search in [Setting > Etc > Search Engine] in admin page.
    Term Click a text box and you'll see a calendar popped up. Set a start date and end date.
    Additional Condition In oder to search faster, you can select another condition among Flagged, Attached, Read and Unread.

  3. Click Search.

[NOTE]
If you click a button() next to Search, window will go to the previous page.
