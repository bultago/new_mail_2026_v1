Add/Modify/Delete Contacts

You can add, modify and delete your contacts in Contacts.

Add Contacts

To add contacts, follow the steps below.

  1. Click Contacts on top and select a group.
  2. Click Add in Basic tab menu.
  3. Contact window will be prompted. Enter new contact information.
    Item Description
    Basic Information Enter basic contact information.
    • Last Name (*), First Name(*), Middle Name
    • Display Name - Enter a display name that appears in the contacts list.
    • E-mail(*)
    • Nick Name , Birthday, Anniversary, Mobile Number
    Home Enter home information.
    Business Enter business information.

  4. Click Submit once you finish entering information.
  5. The completion message will be popped up. Click OK.

Modify Contacts

  1. Click Contacts on top and select a group.
  2. Select a contact and click Modify in Basic tab menu.
  3. Contact window will be prompted.
  4. Click Submit once you finish editing information.
  5. The completion message will be popped up. Click OK.

Delete Contacts

  1. Click Contacts on top and select a group.
  2. Select a contact.
  3. Click Delete in Basic tab menu.
  4. The confirmation message will be popped up. Click OK.

 
Move/Copy Contacts

You can copy or move your contacts to other groups.

Move Contacts

  1. Click Contacts on top and select a group.
  2. Select a contact.
  3. Click Move in Basic tab menu.
  4. Group list will appear. Select a group.

Copy Contacts

  1. Click Contacts on top and select a group.
  2. Select a contact.
  3. Click Copy in Basic tab menu.
  4. Group list will appear. Select a group.
