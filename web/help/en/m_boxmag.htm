Folder managements provides you with functions as below.

  • Folder Management - default folders, personal folders, search folders
  • Tag Management
  • Share personal folders
  • Download Messages, Upload Messages

You can go to manage folders as follows.

  • Click Manage next to message list in [Mail].
  • Click [Options > Folder Management].

Below describes how to manage folders in [Folder Management].

 
Default Folder Management

You can confirm how many messages are in default folders and how much storage they are using. Default folders are provided by default by the system so folders cannot be added, modified and deleted.

Items Description
Folder

Folder names are displayed.

Count The number of unread messages/the number of total messages are displayed.
Storage Usage It shows how much storage folders are using.

 

Empty Default Folder

It enables you to empty folders at once.

  1. EDIT will appear once you mouse over each folder of default folders in [Folder Management].
  2. Click EDIT.
  3. Click Empty on a sub menu.

Download

You can compress folders and download them to your PC.

[NOTE]
Refer to Download Message to learn how to backup folders.

 
Personal Folder Management

This helps you to make folders as your preference and manage them easily.

Items Description
Folder

Folder names are displayed.
Aging Period

You can set aging period for each folder. Messages will be deleted forever if they are stored over the aging period.

  • Disabled, Unlimited, 30 Days, 90 Days, 120 Days
Count The number of unread messages/the number of total messages are displayed.
Storage Usage It shows how much storage folders are using.

To add, modified and delete personal folders, follow the steps below.

[NOTE]
You can manage personal folders in [Mail]. For more information, refer to Personal Folder Management.

Add Personal Folder

To create personal folders, follow the steps below. You can make them up to two depths.

  1. Type a new folder name next to Personal Folders in Folder Management.
  2. Click Add.

If you'd like to create a sub folder,

  1. EDIT will appear once you mouse over a folder you'd like to add in Folder Management.
  2. Click EDIT.
  3. Click Add on a sub menu.
  4. Type a folder name and click Add.

Edit Personal Folder

Personal folder names can be edited.

  1. EDIT will appear once you mouse over a folder you'd like to edit in Folder Management.
  2. Click EDIT.
  3. Click Modify on a sub menu.
  4. Enter a new folder name and click Modify.

Empty Personal Folder

It enables you to empty your personal folders at once.

  1. EDIT will appear once you mouse over each folder of your personal folders in Folder Management.
  2. Click EDIT.
  3. Click Empty on a sub menu.

Delete Personal Folder

[NOTE]
Once a folder is deleted, messages in the folder cannot be recovered.

  1. EDIT will appear once you mouse over a folder you'd like to delete in Folder Management.
  2. Click EDIT.
  3. Click Delete on a sub menu.
  4. A completion message will be prompted. Click OK.

Share Personal Folder

You can share your messages with other users once you share your personal folders. If you assign users who you will share your messages with, they can read messages in the shared folders.

[NOTE]
Refer to Share Personal Folder to learn more about how to share personal folders.

Upload Messages

You can upload messages which are in your PC into your personal folders.

[NOTE]
To d messages, refer to Upload Message.

Download Message

You can backup messages by compressing a folder and download it into your PC.

[NOTE]
For more information about how to backup folders, refer to Download Message.

 

 
Shared Folders

You can see that shared folders that both you shared and other users shared in Personal Folders.

[NOTE]
Refer to Share Personal Folder to learn more about how to share personal folders.

 
Search Folder Management

Search folder makes you save your queries and shows results in the search folders. It doesn't create folders physically but logically so it shows messages which are matched with your queries.

[NOTE]
Search Folder Management can be managed in [Mail]. For more information, refer to Search Folder Management.

Add Search Folder

To add a search folder, follow the steps below.

  1. Select or enter items below..
    Items of Search Folders are as below.
    • SearchFolder Name - Type a new search folder name.
    • From - Type a sender's email address.
    • To (Include TO/CC) - Enter a recipient's email addresses.
    • Query - Select a condition and type into a keyword.
    • Folder - Select a folder where you'd like to search.
    • Additional Condition - In oder to search faster, you can select another condition among Flagged, Attached, Read and Unread.
  2. Click Add.

Modify Search Folder

To modify search folders, follow the steps below.

  1. EDIT will appear once you mouse over a folder you'd like to modify in Manage Search Folders in Folder Management.
  2. Click EDIT.
  3. Click Modify on a sub menu.
  4. SearchFolder Setup window will be popped up. Edit search folder information.
  5. Click Submit.

Delete Search Folder

You can delete search folders. Even though you delete search folders, messages in the folders are not deleted.

  1. EDIT will appear once you mouse over a folder you'd like to delete in Manage Search Folders in Folder Management.
  2. Click EDIT.
  3. Click Delete on a sub menu.
  4. A completion message will be prompted. Click OK.

 
Tag Management

You can assign different tags to received messages. Once you assign tags, you can read messages having the same tag at once.

[NOTE]
You can add, modify and delete tags in [Mail]. For more information, refer to Tag Management.

Create Tag

To add tags, follow the steps below.

  1. Click Add in Manage Tag in Folder Management.
  2. Tag Setup window will be popped up.
  3. Enter Tag Name and select Tag Color.
  4. Click Submit. If you'd like to cancel, click Close.

Edit Tag

To edit tags, follow the steps below.

  1. EDIT will appear once you mouse over a tag you'd like to edit.
  2. Click EDIT.
  3. Click Modify on a sub menu.
  4. When Tag Setup window appears, enter a new tag name and change a color.
  5. Click Submit. If you'd like to cancel, click Close.

Delete Tag

To delete tags, follow the steps below. Once you delete a tag, messages with the tag do not have the tag.

  1. EDIT will appear once you mouse over a tag you'd like to delete.
  2. Click EDIT.
  3. Click Delete on a sub menu.
  4. A completion message will be prompted. Click OK.

 
Share Personal Folder

It enables you to share your personal folders with other users. If you'd like to share your personal folders, assign users who can read messages in the shared folders as they sign on.

Once you share folders, they will appear with Shared Folder Icon().

[NOTE]
You can share your personal folders in [Mail]. For more information, refer to Share Personal Folder.

Personal Folder Share Configuration

To share your personal folders, follow the steps below.

  1. EDIT will appear once you mouse over a folder you'd like to share in Folder Management.
  2. Click EDIT and click Shared on a sub menu.
  3. Shared Folders window will appear. Enter items below.
    • Settings - Select Use.
    • Search - Enter either ID or user name and click Search.
  4. Select users you'd like to add to share from the result and click Add.
  5. Click Submit.
  6. A completion message will be prompted. Click OK.
  7. You can see that the folder icon turned into shared folder icon().

Cancel Folder Sharing

To cancel sharing, follow the steps below.

  1. EDIT will appear once you mouse over a folder you'd like to cancel sharing in Folder Management.
  2. Click EDIT and click Shared on a sub menu.
  3. Shared Folders window will appear.
  4. Click Not use on Settings.
  5. Click Submit.
  6. A completion message will be prompted. Click OK.

 

 
Download Message

You can backup your messages to your PC by compressing a folder and download it. To backup messages, follow the steps below.

  1. EDIT will appear once you mouse over a folder you'd like to backup in Folder Management.
  2. Click EDIT and click Download.
  3. Progress of compression will be displayed on the top of the screen.
  4. A completion message will appear once compression is finished.
  5. Click Download backup file to download the compressed folder messages as below.
    • Delete backup file - Click Delete backup file to delete the compressed file. The compressed file should be deleted if you'd like to download another folder.
       

                 [Backup file download]
  6. Download window will appear. Click Save.
  7. Select a directory to save and click Save.

 
Upload Message

You can upload your messages in your PC into your personal folders in TMS. To upload messages, follow the steps below.

  1. EDIT will appear once you mouse over a folder you'd like to upload into in Folder Management.
  2. Click EDIT and click Upload on a sub menu.
  3. Upload window will appear. Click Browse.
  4. Select files to upload in your PC and click Open.
  5. A window which shows progress will appear.
  6. A completion message will be prompted. Click OK.
