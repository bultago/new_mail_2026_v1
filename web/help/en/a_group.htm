Add/Edit/Delete Groups

You can be able to add, edit and delete groups. In case of shared contacts, group management is based on access authority.

[NOTE]
For more information on access authority for contacts, please refer to Access Authority.

Add Group

You can add groups to either your contacts or shared contacts.

[NOTE]
When you'd like to add groups to Shared Contacts, you have to create the root contact beforehand. You are able to add the root contact in [Service> Domain List> Contacts Management> Shared Contacts] in web admin page.

  1. Click Contacts on top and Add next to Groups on the left.
  2. Group window will be popped up.
  3. Enter a group name.
  4. Click Submit.

Edit Group

  1. Click Contacts on top and Manage next to Groups on the left.
  2. Group window will be popped up.
  3. Select a group name to edit and enter a new group name.
  4. Click Submit.
  5. The completion message will appear. Click OK.

Delete Group

  1. Click Contacts on top and Manage next to Groups on the left.
  2. Group window will be popped up.
  3. Select a group name to delete.
  4. Click Delete.
  5. The completion message will appear. Click OK.

 
Assign Access Authority

This describes how to give access authority of shared contacts.

[NOTE] You can give access authority as long as you are Creator or Moderator of the groups you'd like to give.


To give access authority, follow the steps below.

  1. Click Contacts on top and Shared on the left.
  2. Select a group you'd like to edit.
  3. Click Moderator or Viewer in More tab menu.
  4. Moderator or Viewer window will be prompted.
  5. Enter ID you'd like to give authority and click Add.
  6. Click Close.
