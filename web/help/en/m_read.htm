Read Messages

Incoming messages go to Inbox folder by default.

If you click one of your mail boxes, a message list in the box will be displayed. A message list consists of items as below.

[NOTE]
According to your skin, menus will be displayed in a page without a toolbar or tab menu. Skin can be set in [Services > Domains > Webmail Settings > Skin Management] by your administrator.

Item Description
Checkbox Select checkboxes in order to delete, copy or move messages from the list.
Sorting Type

Show status of incoming messages.

  • Flag message(, ) - Add/Remove flag to be classified as important messages or personal ones. Add/Remove flag by clicking the flag icon.
  • Read/Unread(, ) - Messages can be classified according to Read/Unread status.
  • Attached File() - Shows there is an attached file.
     

To sort mails, follow the steps below.

  • Click [Basic>Quick Search]. Select a sorting type in a message list.
  • Click a sorting type in Search Folders.
    • Flagged, Attached, Unread, Read, Yesterday, Today
From/To

A sender or recipient's email address or name is shown.
If you click the email address or name, sub menu will be displayed. You can write an email or add the address to contacts right away through the sub menu.

  • Compose – write an email to the sender or recipient.
  • Add to Contacts – add the address to your contacts.
  • Search Messages – search messages which are matched with From or To field.
CC Mark

If you received a message as CC, CC icon() will be displayed before a subject. This helps you read important messages first because you know which message are received as CC at a glance.

[NOTE]
Only when CC Mark in [Services > Webmail Settings > Default display settings] in admin page is checked, CC icon() is displayed. If you cannot see the CC icon, please ask your administrator for help.
Subject

Subject of the message. Message contents will be popped up if an icon() next to the subject is clicked.
Date

Shows received or reserve date and time.
Size

Shows message size.

 

View Message Contents

There are three ways to view message contents.

  • View message contents in the current window
  • View message contents in another window
  • Preview message contents

 

[View message contents in the current window]

  1. Click subject that you'd like to view in a message list.
  2. Message contents will be displayed in the current window.

 

[View message contents in another window]

  1. Click an icon() next to subject.
  2. Message contents will be popped up.

 

[Preview message contents]

  1. Select preview pane.

    - Select preview pane

  2. Message contents will be displayed in the right or bottom of the window without opening another window or change the current one once you click subject that you'd like to view in a message list.

Reply / Forward

Reply / Reply to All

You can reply to messages. If you reply, Re: will be written before subject.

  • Reply - Send an email to a sender.
  • Reply to All - Send an email to a sender and all recipients including cc and bcc.

If you want to reply, follow the steps below.

  1. Check a message and click Reply(Reply to All) on the Basic tab menu. Or, click Reply(Reply to All) on the Basic tab menu in message contents.
  2. To will be filled automatically.
    • Reply - To will be filled with only the sender's email address.
    • Reply to All - To will be filled with the sender, cc and bcc's email address.
  3. Click Send once you finish writing a message.

Forward

You can forward messages you received to other accounts.

  • If you forward a message, Fwd: will be written before subject.

If you want to forward a message, follow the steps below.

  1. Check a message and click Forward on the Basic tab menu. Or, click Forward on the Basic tab menu in message contents.
  2. Select one of items below.
    • Source Attached - When forwarding a message, the message will be included as an attachment file.
    • Source Parsed - When forwarding a message, the message will be included in a message body part.

    3. [NOTE]
    If you choose more than a message in a message list and click Forward, the messages that you selected will be sent as only Source Attached.

  3. Click Send once you finish writing a message.

If you want to move/copy a message, follow the steps below.

  • Click Move or Copy on the Basic tab menu.
  • Drag and drop messages to move.

Click Move or Copy on the Basic tab menu.

Select messages from a message list.

    1. Check messages and click Move or Copy. Or, click subject of a message and clickMove or Copy on the Basic tab menu of message contents.
    2. Mail box will be popped up like below.
        
    3. Select a mail box.
    4. Messages will be moved or copied to the mail box you chose.

Drag and drop

Drag and drop messages to move. Make sure that you cannot copy messages by drag and drop.

Delete

Delete messages in mail boxes.

  • Delete - Messages go to Trash.
  • Delete Forever - Messages are deleted forever and cannot be recovered.

Delete

If you want to delete a message, follow the steps below.

  1. Check messages that you'd like to delete. Or, click subject of a message.
  2. Click Delete on the Basic tab menu.
  3. The messages move to Trash.

Delete Forever

If you want to delete a message forever, follow the steps below.

  1. Check messages that you'd like to delete. Or, click subject of a message.
  2. Click Delete Forever on the Basic tab menu.
  3. Click OK once a confirmation message is prompted.

 
Print
  1. Check a message that you'd like to print. Or, click subject of a message.
  2. Click Print on the Basic tab menu.
  3. Click OK once a confirmation message is prompted.

 
Resend

You can send an email which you have already sent and saved in Sent. You can modify To, Cc, Bcc, Subject and etc when you resend an email.

  1. Select a message you'd like to resend in Sent and click Resend on the More tab menu.
    Or, click the subject of the message you'd like to resend. The window will go to Compose. Click Resend on the More tab menu.
  2. To, Cc, Bcc, subject and attachments are automatically filled in.
  3. Click Send once you confirm all the information of the message.

 
Mark as Read/Unread

Mark received messages as read or unread so that messages can be sorted by this status.

  • Read() - Mark as read.
  • Unread() - Mark as unread.

If you want to mark messages as read or unread, follow the steps below.

  1. Check messages that you'd like to mark. Or, click subject of a message.
  2. Click Read/Unread on the More tab menu.
  3. Click Read or Unread.

[NOTE]
It is not possible to mark messages as unread in case of messages that you replied or forwarded.

 
Report Spam

Remove unwanted messages from your Inbox. You can prevent unsolicited messages from entering your Inbox once you report spam.

  1. Check messages that you'd like to mark as spam. Or, click subject of a message.
  2. Click Report Spam on the Basic tab menu.
  3. Check spam options.
    • Add the sender to the blocking addresses - Report as spam to the administrator. The sender's email address will be added to the blocking addresses so that messages from the sender will be prevented from entering your Inbox.
    • Move this message to Trash - Report as spam to the administrator and move the messages to Trash. You can restore it later.
  4. Click Submit. Or, if you want to cancel to report, click Close.

 

 
Allow / Block Addresses

You can add email addresses to allowed and blocking addresses.

[NOTE]
Email addresses to be allowed or blocked are listed in [Options> Spam Management> Manage Unblocked/Blocked Mail Addresses].

Block

Add sender's email addresses from Inbox messages to blocking addresses.

  1. Click Inbox.
  2. Check messages to prevent from entering your Inbox and click Reject Sender on the More tab menu.
  3. A confirmation message will be prompted.
  4. Click OK if you'd like to add. If you'd like to undo, click Cancel.
  5. A completion message will be prompted. Click OK if you want to delete it.

Allow

Add sender's email addresses from messages in Spam or Trash to allowed addresses.

  1. Click Spam or Trash.
  2. Check messages to accept and click Allow Sender.
  3. A confirmation message will be prompted.
  4. Click OK if you'd like to add. If you'd like to undo, click Cancel.
  5. A completion message will be prompted. Click OK if you want to move to Inbox.

 
Quota Excess

A warning message, Quota violate, displays on the top of message contents when you're over your mail box quota. It is recommended to delete unnecessary messages.

There are some restrictions when you're over the quota.

  • It is unavailable to send a secure message and to return receipt.
  • Sent messages are not saved.
  • It is unavailable to send a message with a scheduled delivery.
  • Draft messages are not auto saved.

[NOTE]
Trash and Spam are not included in quota.

[NOTE]
Administrator can set what action will be taken if users exceed their mail box quota. For more detail information, please refer to Web Administrator's Guide.

 
Import Mail

You can import messages from other mail servers using POP3 and POPS. Your other accounts should be added in [Options> Import Mail].

[NOTE]
Refer to [Options > Import Mail] to learn about how to set your other email accounts.

If you want to import messages from other accounts, follow the steps below.

  1. Click Import Mail on the left menu.
  2. Another window will be popped up.
  3. Select email accounts that you'd like to download messages and click Get POP3 mail.

 
Archive Center

This menu appears if Terrace Mail Suite is compatible with Terrace Mail Archive provided by Daou Tech Inc. You can see your inbound and outbound messages archived via Archive Center without any authentication.

[NOTE]
To use this function, Archive Center in [Services > Domains > Account > Group Management] should be enabled. If you'd like to change TMA settings, go to [Setting > Etc > Linking to TMA] in admin page.

To view archived messages, follow the steps below.

  1. Click Archive Center on the left menu.
  2. Archive Center window will be prompted.
  3. View inbound and outbound messages.
