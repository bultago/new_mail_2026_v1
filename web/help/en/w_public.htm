Add a Shared Web Folder

Add the Shared Web Folder to a shared folder for other users.

  1. Click the [Web Folders > My Web Folder].
  2. In the Extra tab of toolbar, click the Search Shared Folder button.
  3. Search Shared Folder window appears.
  4. Enter user ID or name in the search term, and then click Search Shared Folder.
  5. In the search results list, select the item to be added to the shared folder.
  6. Click the OK button.
  7. Completion message is displayed. Click the OK button.

 
Remove Shared Web Folders

Add a shared folder on a Shared Web Folder.

  1. Go to the Shared Web Folder in the left frame of the Web Folders.
  2. Shared Web Folder from the tree, click the folder you want to delete.
  3. In the Extra tab of toolbar,Click the Sharing Off.
  4. Disable the confirmation message window appears.
  5. To disable the shared folder OK button.
  6. Message appears in the complete release. Click the OK button.
