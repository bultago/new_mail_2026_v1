メールフォルダ管理

フォルダを追加してメールの管理ができます。

フォルダの追加

フォルダを追加します。追加は2階層まで追加ができます。

  1. フォルダ管理 画面の 追加をクリックします。
  2. フォルダ名を入力します。
  3. ボタンをクリックします。

追加したフォルダ下位にメールフォルダを追加する場合は

  1. フォルダ管理画面のフォルダで追加する上位フォルダにマウスのポインタをあわせるとEDITが表示されます。
  2. EDITをクリックします。
  3. 下位メニューリストで 追加をクリックします。
  4. フォルダ名を入力した後、 ボタンをクリックします。

フォルダの変更

フォルダ名を変更します。

  1. フォルダ管理画面のフォルダで変更するフォルダにマウスのポインタをあわせるとEDITが表示されます。
  2. EDITをクリックします。
  3. 下位メニューリストで 変更をクリックします。
  4. フォルダ名を入力した後、 変更ボタンをクリックします。

フォルダの削除

フォルダを削除します。

[参考]
フォルダを削除すると、削除するフォルダに保存されているメールも削除され復旧することができません。

  1. フォルダ管理画面のフォルダで削除するフォルダにマウスのポインタをあわせるとEDITが表示されます。
  2. EDITをクリックします。
  3. 下位メニューリストで 削除をクリックします。
  4. 確認メッセージが表示されます。確認 ボタンをクリックします。

フォルダを空にする

各フォルダにあるメールを一度に削除することができます。

  1. フォルダ管理画面で、空にするフォルダにマウスのポインタをあわせるとEDITが表示されます。
  2. EDITをクリックします。
  3. 下位メニューリストで空にするをクリックします。

フォルダの共有

フォルダを他の使用者と共有して該当メールフォルダにあるメールを共有します。

 

 
検索フォルダの管理

検索フォルダは使用者が定義した検索条件でフォルダを生成すると、該当検索条件に一致する結果を表示する仮想フォルダです。

検索フォルダを追加・変更・削除する方法は次の通りです。

検索フォルダの追加

  1. 検索フォルダの追加をクリックします。
  2. 検索条件項目を入力します。
    • フォルダ - 検索フォルダ名を入力します。
    • フォルダ選択 - 検索対象のフォルダを選択します。
    • 検索条件 - メールを検索する検索条件を選択します。
      • 件名、内容、添付ファイル名、添付ファイル内容、件名+内容、件名+添付ファイル名、件名+添付ファイル内容
    • 検索キーワードを入力します。
  3. 確認ボタンをクリックします。

検索フォルダの変更

検索フォルダの検索条件を変更します。

  1. 検索フォルダ変更するフォルダにマウスのポインタをあわせるとEDITが表示されます。
  2. EDITをクリックします。
  3. メニューリストで 変更をクリックします。
  4. 検索フォルダ名及び検索条件を修正します。
  5. 変更後、確認ボタンをクリックします。

検索フォルダの削除

検索フォルダを削除します。 検索フォルダを削除してもメールは削除されません。

  1. 検索フォルダ削除するフォルダにマウスのポインタをあわせるとEDITが表示されます。
  2. EDITをクリックします。
  3. メニューリストで 削除をクリックします。
  4. 確認メッセージが表示されます。 確認ボタンをクリックします。

 
共有フォルダ

自分が共有したフォルダと、他のメンバーが共有したフォルダが表示されます。

 
フォルダの共有

フォルダを他のユーザと共有して該当フォルダにあるメールを共有することができます。
共有対象者を指定します。指定された共有対象者は自分のアカウントで登録したウェブメールの共有フォルダリストから共有したフォルダのメールを確認することができます。

フォルダの共有設定

  1. メールメニューの左フレームにあるフォルダ画面で、共有するフォルダにマウスのポインタをあわせるとEDITが表示されます。
  2. EDITをクリックします。
  3. メニューリストで 共有をクリックします。
  4. 共有フォルダの設定ウィンドウが表示されます。
  5. 共有フォルダの設定を 使用するに選択します。
  6. アカウント 又は 名前で共有するメンバーの入力後、 検索 ボタンをクリックします。
  7. 検索結果リストで該当する共有対象者を選択した後、追加 ボタンをクリックします。
  8. 完了後、 確認 ボタンをクリックします。

フォルダの共有解除

  1. メールメニューの左フレームにあるフォルダ画面で、共有を解除するフォルダにマウスのポインタをあわせるとEDITが表示されます。
  2. EDITをクリックします。
  3. メニューリストで 共有をクリックします。
  4. 共有フォルダの設定ウィンドウが表示されます。
  5. 共有フォルダの設定を 使用しないに選択します。
  6. 確認 ボタンをクリックします。
