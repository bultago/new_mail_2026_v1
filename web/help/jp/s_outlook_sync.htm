Outlook同期化は、Webメールの予定と、PCにあるMS Outlookの予定を同期化する機能です。

[参考] Outlook同期化は、Microsoft Windows 2000 以上のOSと、MS Outlook 2003と2007をサポートします。Microsoft Outlook Expressはサポートしません。

 
Outlook同期化モジュールのダウンロード

Outlook同期化機能を使用するためには、Outlook同期化モジュールをPCにダウンロードし、インストールする必要があります。Outlook同期化モジュールのインストール方法は次のとおりです。

  1. [スケジューラ > Outlook同期化] メニューをクリックします。
  2. Outlook同期化モジュールのダウンロードをクリックします。
  3. PCに保存後、該当ファイルを実行します。
  4. Wizard画面の案内にそって、インストールします。
  5. MS Outlookを実行します。
    既に実行されている場合は、プログラムの終了後、実行します。
  6. MS OutlookのメニューバーにTMSのWebメールの同期化メニューがアドインされていることを確認します。

 
同期化設定
MS Outlookの追加された予定を、TMSの同期化設定メニューでWebメールの予定と同期化することができます。

基本情報

同期化設定メニューの基本情報 タブの画面構成は次のとおりです。

  1. ユーザ情報
    Webメールのアカウントとパスワード、そしてWebメールのサーバ名（ホスト名）を入力します。

  2. 基本設定
    MS Outlookの実行時に動作する項目を設定します。
    3. 項 目 説 明
    Outlook実行時、
    自動ログインする

    MS Outlookが実行時に、Webメールに自動ログインするかを設定します。
    Outlook実行時、
    画面を表示する

    MS Outlookが実行時に、Webメールの画面を表示するかを設定します。
    Outlook実行時、
    自動アップグレードする

    同期化モジュールのアップグレードバージョンがある際に、自動でアップグレードするかを設定します。
    同期化完了後、
    ウィンドウを閉じる

    同期化が完了した際に、同期化の表示ウィンドウを閉じるかを設定します。

  3. 同期化方法
    同期化を行う方法を設定します。

    • 手動同期化 – ユーザがTMS同期化をクリックすると、同期化を行います。
    • 自動同期化 – 指定された時間毎に、自動で同期化を行います。
      • 1, 3, 6, 9時間毎に

  4. 同期化リセット
    既に同期化されたものなど、履歴の削除後、最初から同期化を行います。

予定の同期化

同期化設定 メニューの予定同期化 タブの画面構成は次のとおりです。

  1. 同期化方法
    同期化方式と開始日を設定します。
    2. 項 目 説 明
    同期化方式

    Webメールの予定と、MS Outlookの予定の、同期化の方向（片方向、双方向）を設定します。

    • TMSの予定 <-> Outlookの予定 - Webメールの予定と、Outlookの予定を、双方向で同期化しアップデートします。

    • TMSの予定 -> Outlookの予定 - Webメールの予定をOutlookの予定にアップデートします。

    • TMSの予定 <- Outlookの予定 - Outlookの予定をWebメールの予定にアップデートします。
    開始日

    同期化を開始する予定の基準日を設定します。

    • 本日を含めた以降の予定 - 本日の予定を含めた本日以降の予定を同期化します。

    • 同期化モジュール設置後の予定 - Outlook同期化モジュールをインストールしてから登録された予定を同期化します。

  2. 同期するOutlookのフォルダ設定
    Outlookで同期化するフォルダを設定します。

  3. 同期化衝突時のオプション
    TMSとOutlookで同時に変更した予定を同期化する際に衝突が発生します。

    • 衝突発生時の通知 - 衝突が発生した際には、ユーザにどの予定を基準に同期化するかを表示します。ユーザは、同期化する予定を選択することができます。
    • TMSの予定を優先 - 衝突が発生した際には、Webメールの予定を優先し同期化します。
    • Outlookの予定を優先 - 衝突が発生した際には、Outlookの予定を優先し同期化します。

  4. 同期化結果の通知
    予定の追加/削除/変更の結果を表示するかを設定します。

    • 予定の削除時に通知 - 予定が削除された場合、別ウィンドウで結果を表示します。
    • 定の追加、変更時に通知 - 予定が追加、変更された場合、別ウィンドウで結果を表示します。

同期化結果

同期化設定メニューの同期化結果 タブ画面の構成は次のとおりです。

  1. ラスト同期化時間
    直近の同期化した日付と時間を表示します。

  2. 同期化結果の検索

    • 期間設定 - 検索する日付を設定します。
    • 同期化方式 - 検索する同期化方式を設定します。
      • TMSの予定 <-> Outlookの予定
      • TMSの予定 -> Outlookの予定
      • TMSの予定 <- Outlookの予定
    • Outlookフォルダ - 検索するOutlookのフォルダを設定します。

  3. 検索結果
    同期化結果の検索で実行した検索結果を表示します。

 
TMS同期化
TMS同期化ボタンをクリックして、同期化を実行します。[同期化設定 > 基本情報 > 同期化方法]で手動同期化を設定した場合に、同期化が行われます。自動同期化を設定した場合は、同期化をしません。

[参考] 自分が登録した予定は同期化されますが、他のユーザが登録した共有予定は同期化されません。

 
TMSへ移動

同期化設定で設定した情報をベースにWebメールのスケジューラの画面に移動します。
