掲示物の確認

掲示板を通じて告示事項、FAQ及び、掲示物の内容を確認することができます。

  1. [掲示板]メニュー画面の左フレームで確認する掲示板を選択します。
  2. 掲示物リストで掲示物のタイトルをクリックします。

掲示物の構成は次の通りです。

項目 説明
タイトル 掲示物のタイトルです。
掲示物情報

掲示物の基本情報です。

  • 作成者 – 掲示物を作成した作成者の名前
  • 番号 – 該当掲示板に登録された掲示物番号
  • 閲覧数 – 掲示物を確認した数
  • 作成日時 – 掲示物を作成した日時
添付ファイル 掲示物作成時、追加した添付ファイルです。
掲示物 掲示物の内容です。

 
掲示物の転送

掲示板に登録された掲示物をメールで転送します。

  1. [掲示板]メニューの左フレームで確認する掲示板を選択します。
  2. 掲示物リストで掲示物のタイトルをクリックします。
  3. 内容のツールバーで転送をクリックします。
  4. 該当掲示物を含むメール作成画面に切り替わります。
  5. タイトル入力及び、受信者を指定します。
  6. ツールバーで送信ボタンをクリックしてメールを送信します。

 
掲示物を別ウィンドウで表示

掲示板に登録された掲示物を別ウィンドウで表示します。

  1. [掲示板]メニュー画面の左フレームで該当掲示板を選択します。
  2. 掲示物リストでタイトルをクリックします。
  3. 内容のツールバーで別ウィンドウをクリックします。
  4. 別ウィンドウが表示されたら、該当する掲示物を確認します

 
掲示物の印刷

掲示板に登録された掲示物を印刷します。

  1. [掲示板]メニュー画面の左フレームで該当掲示板を選択します。
  2. 掲示物リストでタイトルをクリックします。
  3. 内容のツールバーで印刷をクリックします。
  4. 印刷プレビュー画面が表示されます。印刷ボタンをクリックします。
  5. 印刷画面で印刷ボタンをクリックします。
