メールの確認、作成の基本環境を設定します。

  1. [環境設定 > 基本環境] メニューをクリックします。
  2. メールの基本環境を設定します。
    • 環境設定
    • メール確認設定
    • メール作成設定
  3. 設定の完了後、保存ボタンをクリックします。
    初期設定値に指定したい場合にはデフォルト値にボタンをクリックします。

 
環境設定

メール環境設定項目は次の通りです。

項目 説明
表示言語

Webメール画面の言語を選択します。

  • 日本語、英語、韓国語
1ページに
表示する件数

ページ当たり表示するリストの件数を選択します。

  • 15件、20件、25件、 30件、50件、80件
新着メールの
通知間隔

新しいメールを受信した際、お知らせするお知らせメッセージの使用可否と間隔を選択します。

  • 使用しない、1分、5分、10分、30分、1時間
ウェブユーザインターフォース

Terrace Mail Suiteのウェブユーザインターフォースを選択します。

  • HTML
  • AJAX - ページの転換なしでデータを持ってくる為、処理速度が早くなりますが、一部のブラウザではサポートできない場合もあります。
検索範囲設定

メールリストでメールを検索する場合、検索範囲をスパムフォルダとゴミ箱を含むかどうかを設定します。
アップロードActiveX

メール作成時に、添付ファイルをアップロードする際に、ActiveXを使用するかを設定します。ActiveXを使用すると、メール作成画面でファイルのアップロード関連のActiveXプログラムをインストールする必要があります。AcitveXのアップロードプログラムを利用すると添付ファイルのリンクのみが送信できる大量メール送信機能が利用できます。

[参考] Web管理画面[環境設定 > その他 > 添付ファイル/自動ログアウト]の添付ファイルのアップロード時Active Xの使用可否が使用するに設定された場合のみアップロードActiveXメニューが表示されます。

[参考] ActiveXはMS Internet Explorerブラウザのみ使用できます。

 

 
メール確認設定

メール確認の環境設定項目は次の通りです。

項目 説明
イメージの表示

メール内容にイメージがある場合、イメージを基本的に表示するかどうかを設定します。

[参考] メールの本文にイメージを挿入したメールのみ適用されます。イメージを添付ファイルにしたメールは適用されません。
HTMLタグの表示

セキュリティ上、危険性があるタグの非活性化可否を設定します。
広告やフィッシングの目的でメールに任意に挿入したスクリプトやHTMLタグです。
遮断タグリストは次の通りです。

  • HTML, HEAD, META, BODY, SCRIPT, IFRAME, EMBED

 

 
メール作成設定

メール作成の環境設定項目は次の通りです。

項目 説明
送信フォルダに保存

送信メールを送信フォルダに自動保存可否を設定します。

  • 保存する、保存しない
開封確認機能

送信メールの開封確認可否を設定します。

  • 開封確認する、開封確認しない

[参考] 開封確認機能はHTMLモードでのみサポートします。
エディタモード

メール作成画面のエディタモードを設定します。

  • HTML, TEXT
メール作成方法

メールを基本ウィンドウで作成するか、別ウィンドウで作成するかを設定します。
送信メールの
エンコード

エンコード方式を設定します。

  • 日本語(ISO-2022-JP)、英語(US-ASCII)、韓国語(EUC-KR)、簡体字中国語(GB2312)、繁体字中国語(BIG5)、Unicode(UTF-8)
下書きフォルダの
メールお知らせ

メール作成ボタンをクリックする際に、下書きフォルダにメールがあると確認メッセージを表示します。
ニックネーム

メール送信時、受信者に表示される送信者の名前を設定します。
(例)人事担当者
返信メールアドレス

送信したメールに対する返信を貰う際に、他の受取りたいメールアドレスを入力します。
(例) user@daou.co.jp

名刺添付

メール送信時、メール内容に名刺(V-card)の挿入可否を設定します。

  • 私の名刺管理 – 私の名刺を管理する画面に移動します。
