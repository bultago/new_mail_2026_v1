ファイルをアップロード及び、ダウンロードしてウェブフォルダを利用することができます。また、ウェブフォルダを共有して共有ウェブフォルダの権限によって、確認、編集できる機能を提供します。

 
ウェブフォルダ管理 

ウェブフォルダのファイルをグループ化して管理するため、フォルダを追加、変更、削除することができます。

フォルダの追加

ウェブフォルダのフォルダを追加します。フォルダ追加の4段階まで生成されます。

  1. [ウェブフォルダ]画面の左フレームでフォルダを生成する上位フォルダをクリックします。
  2. メイン画面のツールバーの基本機能タブでフォルダ生成をクリックします。
  3. フォルダ生成画面が表示されます。.
  4. フォルダ名を入力した後、確認ボタンをクリックします。
    • フォルダ名は英字40字、日本語20字以内が入力出来ます。

フォルダの変更

フォルダのフォルダ名を変更します。

  1. [ウェブフォルダ]画面のファイルリストで変更するフォルダをチェックします。
  2. ツールバーの基本機能タブでフォルダ変更をクリックします。
  3. 変更するフォルダ名を入力した後、確認ボタンをクリックします。
  4. 変更完了のメッセージが表示されます。確認ボタンをクリックします。

フォルダの削除

ウェブフォルダにあるフォルダを削除します。

  1. [ウェブフォルダ]画面のフォルダリストで削除するフォルダ(ファイル)をチェックします。
  2. ツールバーの基本機能タブで削除をクリックします。
  3. 削除の確認メッセージが表示されます。確認ボタンをクリックします。.
  4. 削除完了のメッセージが表示されます。確認ボタンをクリックします。

 

フォルダ削除の際、次に注意して下さい。

  • 削除するフォルダに下位フォルダがある場合 – 下位フォルダを削除した後、削除処理を行います。
  • 削除するフォルダを共有フォルダの場合 – 共有解除後、削除処理を行います。
  • 削除するフォルダにファイルがある場合 – フォルダにあるファイルが全て削除されます。

 
フォルダ(ファイル)の移動/コピー

ウェブフォルダのフォルダ(ファイル)を移動及びコピーします。

フォルダ(ファイル)の移動

フォルダ及び、ファイルを移動する方法は次の通りです。

  1. [ウェブフォルダ]画面のフォルダリストで削除するフォルダ(ファイル)をチェックします。
  2. ツールバーの基本機能タブで移動をクリックします。
  3. ウェブフォルダのフォルダ(ファイル) ツリーが表示されます。移動するフォルダ(ファイル)を選択します。
  4. 移動完了のメッセージが表示されます。確認ボタンをクリックします。

フォルダ(ファイル)移動時の仕様はは下記の通りです。

  • 移動するフォルダに下位フォルダ又はファイルを全て含めて移動
  • 共有設定されたフォルダを移動する場合、共有解除した後に移動
  • 移動するフォルダ又は、ファイルと同じファイル名がある場合には移動をキャンセル
  • フォルダ移動時、自分のフォルダ名を選択すると移動をキャンセル
  • 最下位フォルダを選択した場合は移動をキャンセル
  • 該当共有ウェブフォルダの読み/書きの権限がある場合、ウェブフォルダのフォルダまたは、ファイルを共有フォルダに移動可能
  • 該当のパブリックフォルダの読み/書きの権限がある場合、ウェブフォルダのフォルダ又は、ファイルをパブリックフォルダに移動可能

 

フォルダ(ファイル)のコピー

フォルダ及び、ファイルをコピーする方法は次の通りです。

  1. [ウェブフォルダ]のフォルダ(ファイル) リストでコピーするフォルダ(ファイル)をチェックします。
  2. ツールバーの基本機能タブでコピーをクリックします。
  3. ウェブフォルダのフォルダ(ファイル) ツリーが表示されます。コピーするフォルダ(ファイル)を選択します。
  4. コピー完了のメッセージが表示されます。確認ボタンをクリックします。

フォルダ(ファイル)コピー時の仕様は次の通りです。

  • コピーするフォルダ(ファイル)と同じ名前のフォルダ(ファイル)がある場合にコピーをキャンセル
  • 最下位フォルダ(ファイル)を選択した場合にコピーをキャンセル
  • 該当共有ウェブフォルダの読み/書き権限がある場合、ウェブフォルダのフォルダ又は、ファイルを共有ウェブフォルダにコピーができます。
  • 該当パブリックフォルダの読み/書き権限がある場合、ウェブフォルダのフォルダ又は、ファイルをパブリックフォルダにコピーができます。
  • 共有ウェブフォルダの読み権限のみあっても、共有ウェブフォルダのフォルダ及ファイルをウェブフォルダにコピー可能。
  • パブリックフォルダの読み権限のみあっても、パブリックフォルダのフォルダ及び、ファイルをウェブフォルダにコピー可能。

 
ファイルのメール送信

ウェブフォルダにアップロードされているファイルを添付してメールを送信することができます。

  1. [ウェブフォルダ]のファイルリストで送信するファイルをチェックします。
  2. ツールバーの基本機能タブでメール送信をクリックします。
  3. 選択したファイルが添付されたメール作成画面が表示されます。
  4. メールを作成した後、送信ボタンをクリックして送信します。

[参考] ウェブフォルダにあるファイルは、既にアップロードされているファイルなので、添付方式（一般添付・大容量添付）の変更ができません。

 
ファイル管理

ウェブフォルダにファイルをアップロード及び、ダウンロード、アップロードしたファイルを削除することができます。

ファイルアップロード

  1. [ウェブフォルダ] 画面の左フレームでファイルをアップロードするフォルダを選択します。
  2. ファイルリスト画面の下段でファイル追加をクリックします。
    • 基本アップロード – 詳細ボタンを通じて、一回で一つのファイルを追加することができます。
    • パワーアップロード – ドラッグ＆ドラッグを通じて、一回で２つ以上のファイルを追加することができます。
    ローカルPCでファイルを選択した後、開くをクリックします。
  3. アップロードをクリックします。

[参考] 同じファイル名をアップロードすると、上書きされずに、別のファイルとしてアップロードされます。

[参考] ウェブ管理コンソールの[フィルタ管理 > ウイルスフィルタ]のウイルス検査が使用の場合、アップロードするファイルのウイルスを検査します。 ファイルをフォルダにアップロードする際、添付ファイルのサイズが[フィルタ管理 > ウイルスフィルタ]の最大検査メールサイズ以内のメールのみウイルス検査を実施します。

ファイルダウンロード

ファイルをダウンロードする方法は次の通りです。

  1. [ウェブフォルダ]画面の左フレームでフォルダを選択します。
  2. ファイルリストでダウンロードするファイル名をクリックします。

    3. [参考] 複数のファイルをダウンロードしたい場合にはダウンロードするファイルをチェックした後、基本機能のタブでファイルダウンロードをクリックします。

  3. 開く又は保存をクリックしてファイルをダウンロードします。

 

ファイル削除

ファイルを削除する方法は次の通りです。

  1. [ウェブフォルダ]画面のフォルダリストで削除するファイルをチェックします。
  2. ツールバーの基本機能タブで削除をクリックします。
  3. 削除の確認メッセージが表示されます。確認ボタンをクリックします。
  4. 削除完了のメッセージが表示されます。確認ボタンをクリックします。

 

 
フォルダ共有管理

フォルダを共有設定して他のユーザにも確認/編集の権限を付与します。他のユーザは共有ウェブフォルダで共有設定したウェブフォルダの確認と編集することができます。

ウェブフォルダ共有設定

フォルダを共有設定する方法は次の通りです。

  1. [ウェブフォルダ] メニューの左フレームでウェブフォルダをクリックします。
  2. ウェブフォルダのフォルダリストで共有するフォルダを選択します。
  3. ツールバーの拡張機能タブでフォルダ共有をクリックします。
  4. フォルダ共有設定が表示されます。各項目を入力して共有を設定します。

    項目 説明
    フォルダ名 共有を設定するフォルダ名です。
    説明 フォルダ共有の目的及び、説明を入力します。
    全体共有

    全体使用者をチェックすると使用者全員にフォルダが共有されます。フォルダ使用権限を選択します。

    • 閲覧権限
    • 閲覧/編集権限
    特定使用者のみにフォルダを共有

    特定使用者のみにフォルダを共有する場合、特定使用者を指定して、権限を付与します。

    1. ID又は名前を入力した後、検索ボタンをクリックして使用者を検索します。
    2. 検索結果リストで使用者を選択した後、追加ボタンをクリックして該当の権限リストに追加します。

    組織図（オプション）が利用できる環境ですと、組織図からも検索が可能です。組織図では、下位組織まで含めて共有することができます。

    5.  
  5. フォルダ共有設定の完了後、確認ボタンをクリックします。
  6. 共有完了のメッセージが表示されます。確認ボタンをクリックします。
  7. 共有を設定したフォルダは共有フォルダアイコン()に変更されます。

フォルダ共有変更

共有設定したフォルダの共有設定情報を変更します。

  1. [ウェブフォルダ]の左フレームでウェブフォルダをクリックします。
  2. ウェブフォルダのフォルダリストで共有したフォルダをクリックします。
  3. ツールバーの拡張機能タブで共有設定確認をクリックします。
  4. フォルダ共有設定が表示されます。設定した共有情報を変更します。
  5. 変更の完了後、確認ボタンをクリックします。
  6. 共有設定完了のメッセージが表示されます。確認ボタンをクリックします。

フォルダ共有解除

共有設定したウェブフォルダを共有解除します。

  1. [ウェブフォルダ]の左フレームでウェブフォルダをクリックします。
  2. ウェブフォルダのフォルダリストで共有したフォルダをクリックします。
  3. ツールバーの拡張機能タブで現在フォルダ解除をクリックします。
  4. 共有フォルダ解除の確認メッセージが表示されます。共有解除する場合には確認ボタンをクリックします。
  5. 共有解除の確認メッセージが表示されます。確認ボタンをクリックします。
  6. 共有が解除されたフォルダはフォルダアイコン()に変更されます。
