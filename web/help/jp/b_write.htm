掲示物の作成

ユーザは該当掲示板に新しい掲示物を作成します。また、掲示板に登録された掲示物の返信又は、コメントを登録することができます。

  • 新しい掲示物の作成
  • 掲示物の返信を作成
  • 掲示物のコメントを作成

新しい掲示物の作成

新しい掲示物を作成します。

  1. [掲示板]メニュー画面の左フレームで掲示板を選択します。
  2. ツールバーの基本機能タブで作成をクリックします。
  3. 掲示物を作成します。
    • タイトル、内容、添付ファイル

    [参考] ウェブ管理コンソールの[フィルタ管理 > ウイルスフィルタ]のウイルス検査が使用の場合、添付ファイルのウイルス検査を実施します。 添付ファイルのサイズが[フィルタ管理 > ウイルスフィルタ]の最大検査メールサイズ以内の場合のみウイルス検査を実施します。

  4. 掲示物の作成の完了後、投稿ボタンをクリックします。
  5. 掲示物の登録のメッセージが表示されます。確認ボタンをクリックします。
  6. 掲示物の登録完了のメッセージが表示されます。確認ボタンをクリックします。

掲示物の返信を作成

掲示板に登録された掲示物のコメントを作成します。コメントを作成すると、掲示物下位にあるコメントアイコン( )と表示され、掲示物が登録されます。

  1. [掲示板]メニュー画面の左フレームで掲示板を選択します。
  2. 掲示物リストでコメントを作成する掲示物のタイトルをクリックします。
  3. 掲示物の内容画面の基本機能タブで返信ボタンをクリックします。
  4. 返信作成画面にてタイトルと内容を作成した後、投稿ボタンをクリックします。
  5. 登録確認のメッセージが表示されます。確認ボタンをクリックします。
  6. 登録完了のメッセージが表示されます。確認ボタンをクリックします。

 
掲示物の変更

掲示板に登録された掲示物を変更します。変更は掲示板管理者と掲示物を作成した本人のみが可能です。

  1. [掲示板]メニュー画面の左フレームで掲示板を選択します。
  2. 掲示物リストで変更する掲示物のタイトルをクリックします。
  3. 内容画面のツールバーで変更ボタンをクリックします。
  4. 変更画面に切り替わります。変更が終わったら、投稿ボタンをクリックします。
  5. 変更確認のメッセージが表示されます。確認ボタンをクリックします。
  6. 変更完了のメッセージが表示されます。確認ボタンをクリックします。

 
掲示物の削除

登録された掲示物を削除します。掲示物削除は掲示板管理者と掲示物を作成した本人のみが可能です。

  1. [掲示板]メニュー画面の左フレームで掲示板を選択します。
  2. 掲示物リストで掲示物をチェックした後、削除ボタンをクリックします。
    また、削除する掲示物のタイトルをクリックして内容画面のツールバーで削除ボタンをクリックします。
  3. 掲示物の削除確認のメッセージが表示されます。確認ボタンをクリックします。
  4. 掲示物の削除完了のメッセージが表示されます。確認ボタンをクリックします。

 
掲示物コメント

掲示物内容下位にコメントを作成し、掲示物とコメントを一度に確認することが可能です。

掲示物コメント作成

掲示物の内容下位にコメントを作成します。

  1. [掲示板]メニュー画面の左フレームで掲示板を選択します。
  2. コメントを作成する掲示物のタイトルをクリックします。
  3. 掲示物画面の下位にコメントを作成した後、投稿ボタンをクリックします。
    • コメント内容は300文字（日本語）まで入力ができます。

掲示物のコメント削除

掲示物に作成されたコメントを削除します。コメントの削除は、権限のある管理者と本人が作成したコメントのみを削除することができます。

  1. [掲示板]メニュー画面の左フレームで掲示板を選択します。
  2. コメントがある掲示物のタイトルをクリックします。
  3. 掲示物画面下位のコメントリストで削除するコメント横の削除ボタンをクリックします。
