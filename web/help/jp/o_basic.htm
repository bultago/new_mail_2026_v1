組織図では、社内組織を階層構造と、組織のメンバー（組織員）が確認できます。

[参考] 組織図はオプション機能です。ライセンスがない場合は、メニューに表示されません。

 
組織図構成

組織図は、次のように構成されています。

  • 部署 - 画面の左側に組織図の部署が階層構造で表示されます。
  • 部署員（組織のメンバー）- 部署をクリックすると、画面の右側に部署員のリストが表示されます。

 
部署員の情報確認

部署員のリスト構成

部署員のリストは、次のように構成されています。

項目 説明
組織名 部署員が所属している部署の名前です。
名前

部署員の名前です。

  • 部署員の名前にマウスのポインタを移動すると、写真が表示されます（該当ユーザがアカウント情報で登録した写真が表示されます）
  • 部署員の名前をクリックすると、詳細情報が表示されます。
    • 写真、名前、メールアドレス、社員番号、携帯電話、部署名、職位、役職
職位 部署員の職位です。
役職 部署員の役職です。
電話番号 部署員の電話番号です。
携帯電話 部署員の携帯電話の番号です。
ID 部署員のIDです。IDをクリックすると、メール作成 ウィンドウが表示されます。

 

部署員リストの並べ替え

デフォルトの並べ替えの順序は、管理者が設定します。 ユーザは、部署員のリストから項目をクリックすると、並べ替えを変更することができます。

また、部署員リストから各項目をクリックすると、並べ替えができます。

  • 組織名、名前、職位、役職、電話番号、携帯電話、メールアドレス

 
部署員の検索

組織図から部署員の名前、メールアドレス（ID）、携帯電話、職位、役職で検索することができます。

部署員の検索方法は、次のとおりです。

  1. [組織図]メニューをクリックします。
  2. 部署員リストの右上にある検索から検索条件を選択します。
    • 名前、アカウント、携帯電話、職位、役職
  3. キーワードの入力後、検索ボタンをクリックします。
