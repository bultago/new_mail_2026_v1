# Phase 5 재작업 프롬프트 - 테스트 및 검증

## 전체 재작업 프롬프트

```
Phase 5의 테스트 및 검증 작업을 재수행해줘.

작업 범위:
1. 회귀 테스트 (14개 주요 기능)
2. 성능 테스트 (로그인/메일목록/메일읽기, 부하 테스트)
3. 보안 테스트 (OWASP Top 10, 침투 테스트)
4. UAT (사용자 승인 테스트)
5. 문서화 (API 문서, 매뉴얼, 가이드)

참조: docs/plans/phase-5/

목표: 회귀 테스트 100% 통과, 성능 목표 달성, 보안 취약점 0건
```

## 성능 테스트만 재실행

```
성능 테스트를 재수행해줘. (P5-015 ~ P5-023)

측정 항목:
- 로그인: < 400ms (20% 개선)
- 메일 목록: < 800ms (20% 개선)
- 메일 읽기: < 600ms (25% 개선)
- 동시 사용자: 100/500/1000명

JMeter 사용, 결과 리포트 생성
```

## 보안 테스트만 재실행

```
보안 테스트를 재수행해줘. (P5-024 ~ P5-035)

테스트 항목:
- SQL Injection
- XSS
- CSRF
- 인증/권한 테스트
- OWASP ZAP 스캔
- 취약점 0건 목표
```
