    암호화
         암호화(인증서 방식) - 인증서 선택 창에서 수신자의 인증서를 선택한다.
         암호화(Password 방식) - 수신자가 알고 있는 패스워드를 입력하면 됩니다.
              적절한 패스워드 문구를 선택 또는 입력하세요.
         암호화( ID/Password 방식) - 이메일 계정이 ID와 아래 입력하신 암호가 같이 사용됩니다.
              패스워드 문구는 위와 동일합니다.
