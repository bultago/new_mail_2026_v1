웹메일 주요 기능

 
웹메일 주요 기능
  • 메일 검색 기능
    메일함에 있는 메일중에서 검색어, 검색대상이 될 항목(제목, 보낸사람)을 지정하여 원하는 메일을 검색할 수 있습니다.

  • 임시 보관함 기능
    사용자가 메일을 작성하다가 부득이한 사정으로 인해 완성하지 못했을 경우에 임시보관함에 저장을 했다가 나중에 다시 불러서 완성하여 보낼 수 있습니다.

  • 예약 발송 기능
    작성한 메일을 지금 바로 발송하지 않고 원하는 시간을 지정하여 그 시간에 발송되도록 하는 기능입니다. 예약시간을 지정하고 [메일보내기]를 클릭하시면 예약발송함에 저장이 되어 있다가 해당 시간이 되면 발송이 됩니다.

  • 자동 분류기능
    받은 메일중 제목이나 보낸사람에 사용자가 지정한 규칙과 일치하면 원하시는 메일함으로 자동 분류하는 기능입니다.

  • 주소록 기능
    메일을 여러명에게 송신할 수 있고, 여러명을 그룹으로 묶어 관리가 용이하게 합니다. 가 ~ 하, A~Z 까지 있으며 등록하면 자동으로 첫 글자의 목록으로 등록되어집니다. Outlook express의 주소록을 가져와 등록할 수 있습니다.

  • 공용 주소록 기능
    도메인내의 사용자들이 공통적으로 사용을 할 수 있는 주소록을 제공합니다. 메일을 여러명에게 송신할 수 있고, 여러명을 그룹으로 묶어 관리가 용이하게 합니다. 가 ~ 하, A~Z 까지 있으며 등록하면 자동으로 첫 글자의 목록으로 등록되어집니다. Outlook express의 주소록을 가져와 등록할 수 있습니다. 또한, 공용주소록의 주소를 개인 주소록으로 옮겨다 사용을 할 수 있습니다.

  • 메일함 관리기능
    메일함의 용량정보 확인 및 추가, 삭제가 가능합니다. 기본으로 제공되는 메일함은 받은 메일함, 보낸메일함, 임시보관함, 휴지통, 스팸메일함이 있습니다. 이 메일함들은 삭제 및 이름 변경이 안됩니다. 그 중에서 스팸 메일함은 환경설정의 스팸메일 차단에 의해서 걸러지는 메일들이 임시 저장되는 메일함입니다. 각 메일함들은 자기 자신의 컴퓨터로 tar로 압축된 백업이 가능합니다. 받은 화일은 outlook express로 열어볼수 있습니다.

  • 자동 분류기능
    받은 메일중 제목이나 보낸사람에 사용자가 지정한 규칙과 일치하면 원하시는 편지함으로 자동 분류하는 기능입니다.

  • 수신확인 기능
    수신상태를 확인하여 메일을 읽었는지 여부와 발송 실패시 실패 사유를 쉽게 알 수 있습니다.

  • 수신 거부 기능
    자신이 원하지 않는 메일이나 특정인에게 오는 메일을 거부할 수 있습니다. 환경설정에서 [수신거부설정]을 클릭하여 설정하면 됩니다.

  • 부재중 응답 설정 기능
    오랫동안 메일을 확인할 수 없을 때 [부재중 응답설정 기능]을 이용하여 자동으로 답변할 수 있게 합니다. 자동 응답할 메일을 따로 설정할 수도 있습니다.

  • 서명 설정 기능
    메일을 보낼 때 자동으로 자신의 프로파일등 서명을 붙일 수 있습니다.

  • 스팸메일 차단
    새로 수신되는 메일중 미리 지정된 스팸메일 걸러내기 단계에 맞추어 스팸메일함으로 자동 분류되는 기능입니다.

  • 휴지통
    메일함에서 지운 편지를 임시 저장하는 메일함입니다.
