자동전달
자동전달 설정 관리

 

자동전달 기능은 수신된 메일을 환경설정에서 지정한 다른 메일 주소로 자동전달하는 기능입니다.

 
자동전달 설정 관리

자동전달 모드 설정 및 규칙 추가

자동 전달 모드 설정 및 자동 전달할 메일을 추가합니다. 자동 전달 규칙은 최대 1000개까지 추가할 수 있습니다.

  1. [환경설정 > 자동전달] 메뉴를 클릭합니다.
  2. 자동 전달의 자동 전달 모드에서 적용 여부를 선택합니다.
    • 사용 안함 - 자동전달 기능을 사용하지 않습니다.
    • 서버에 메일 원본 남겨두기 - 수신된 메일의 원본은 받은 메일함에 저장하고, 메일을 해당 메일 주소로 전달합니다.
    • 서버에서 메일 원본 삭제하기 - 수신된 메일의 원본은 받은 메일함에 저장하지 않고, 바로 다른 메일 주소로 전달합니다.
  3. 기본규칙을 설정합니다. 자동전달 모드를 사용으로 할 경우, 모든 수신 메일은 전달할 메일 주소로 전달됩니다.
    1. 기본규칙 텍스트 박스에 전달할 메일 주소를 입력합니다. 전달할 메일 주소는 전체 메일 주소 또는 도메인을 입력합니다.
      (예) daou_user@daou.co.kr, @daou.co.kr
    2. 추가 버튼을 클릭합니다.
    3. 메일 목록에 메일 주소가 추가됩니다.
  4. 예외규칙을 설정합니다. 예외규칙은 특정한 메일 주소에서 수신된 메일을 전달할 메일 주소록 전달합니다.
    1. 예외규칙 옆에 있는 드롭다운 박스에서 보낸이 주소(From) 또는 보낸이 주소(From)도메인 부분을 선택한 후, 옆에 있는 텍스트 박스에 메일 주소나 도메인을 입력합니다.
    2. 드롭다운 박스 아래에 있는 텍스트 박스에 전달할 메일 주소를 입력합니다. 전달할 메일 주소는 전체 메일 주소 또는 도메인을 입력합니다.
      (예) daou_user@daou.co.kr , @daou.co.kr
    3. 추가 버튼을 클릭합니다.
    4. 메일 목록에 메일 주소가 추가됩니다.

      5. [참고]
      전달할 메일 주소는 웹 어드민의 [서비스 > 계정관리 > 그룹관리]의 최대 자동 포워딩에 설정된 수만큼 추가할 수 있습니다.

  5. 자동전달 설정을 저장하려면 저장하기버튼을 클릭합니다.
  6. 설정 변경 완료 메시지 창이 나타납니다. 확인버튼을 클릭합니다.

 

자동전달 목록에서 규칙 삭제

자동전달 규칙을 목록에서 삭제합니다.

  1. [환경설정 > 자동전달] 메뉴를 클릭합니다.
  2. 기본규칙의 전달 설정한 메일 목록에서 삭제할 메일을 한 개이상 선택합니다. 또는 예외규칙 목록에서 삭제하려는 규칙을 선택합니다.
  3. 삭제버튼을 클릭합니다.
  4. 삭제가 완료되면 저장하기버튼을 클릭합니다.
  5. 설정 변경 완료 메시지 창이 나타납니다. 확인버튼을 클릭합니다.

 

자동전달할 메일주소 검색

자동전달할 메일목록에서 메일 주소를 검색합니다.

  1. [환경설정 > 자동전달] 메뉴를 클릭합니다.
  2. 기본규칙 또는 예외규칙의 전달 설정한 메일 목록에서 검색할 메일 주소를 입력합니다.
  3. 검색버튼을 클릭합니다.
