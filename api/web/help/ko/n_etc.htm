기타 기능

쪽지 목록 위에 기본기능 툴바가 있습니다. 툴바에 있는 버튼을 클릭하여 쪽지를 삭제하거나 장기 보관 또는 수신한 쪽지에 답장을 할 수 있습니다.

쪽지 삭제

쪽지함에 있는 쪽지를 삭제합니다. 쪽지는 삭제되면 복구가 불가능합니다. 삭제하려는 쪽지를 선택한 후 툴바에서 삭제버튼을 클릭합니다.

쪽지 보관

받은쪽지함 또는 보낸쪽지함에 있는 쪽지를 보관할 수 있습니다. 보관하려는 쪽지를 선택한 후 툴바에서 보관버튼을 클릭합니다. 해당 쪽지가 쪽지보관함으로 이동합니다.

[참고]
받은쪽지함과 보낸쪽지함에 있는 쪽지는 웹 어드민의 [서비스 > 계정관리 > 그룹관리]와 [서비스 > 계정관리 > 사용자관리]의 쪽지함 유효기간이 지나면 자동으로 삭제됩니다. 쪽지를 장기보관하고자 할 때는 쪽지보관함에 저장하시기 바랍니다.

[참고]
쪽지는 [서비스 > 계정관리 > 그룹관리]와 [서비스 > 계정관리 > 사용자관리]의 최대 쪽지 보관 수에서 설정한 수만큼 보관할 수 있습니다.

답장

수신된 쪽지의 발신자에게 답장을 발송할 수 있습니다. 받은쪽지함에서 답장하려는 쪽지를 선택한 후 툴바에서 답장버튼을 클릭합니다. 쪽지쓰기 창이 나타나며 받는 사람에 해당 쪽지의 발신자 주소가 자동으로 입력됩니다.

수신 거부

수신된 쪽지의 발신자를 수신거부 목록에 추가합니다. 받은쪽지함에서 수신거부하려는 쪽지를 선택한 후 툴바에서 수신거부버튼을 클릭합니다.

[참고]
[쪽지 > 환경설정]에서 쪽지 차단 정책이 수신 거부 사용자만 거부로 설정되었을 때만 수신거부 목록에 있는 주소에서 수신한 쪽지를 거부합니다.

빠른검색

쪽지함에서 읽은/안읽은 쪽지, 내게 보낸 쪽지를 필터링하여 확인할 수 있습니다. 툴바에서 빠른검색버튼을 클릭합니다. 읽은, 안읽은, 내게 보낸 쪽지의 보기 조건 목록이 나타납니다. 보기 조건 항목을 클릭하면 해당하는 쪽지 목록만 필터링되어 나타납니다.

통합 검색

통합 검색은 보낸이/받는이, 제목의 검색어를 입력하여 검색하는 기능입니다. 쪽지함 이름 우측에 있는 검색란에 검색어를 입력한 뒤 검색버튼을 클릭합니다. 검색결과 목록이 나타납니다.

[참고]
검색 확인 후, 검색입력란에서 를 클릭하면 검색하기 전단계화면으로 이동합니다.

목록개수

한 페이지에 보여지는 쪽지의 개수를 지정할 수 있습니다. 기본기능 툴바 상단에 있는 드롭다운 박스에서 목록개수를 선택합니다.
