쪽지 확인

 

쪽지란 같은 도메인 내 웹 메일 사용자와 주고 받을 수 있는 메모입니다.

[참고]
쪽지 기능은 제품 구입 시 기본적으로 제공되는 기능이 아닙니다. 별도의 구매 후, 사용할 수 있는 기능입니다.

 
쪽지 확인

송수신 쪽지의 보낸(받는)사람, 송수신 날짜, 쪽지 내용을 확인합니다. 쪽지함에는 받은쪽지함, 보낸쪽지함, 쪽지보관함이 있습니다.

쪽지함은 다음과 같이 받은쪽지함, 보낸쪽지함, 쪽지보관함으로 구성되어 있습니다.

받은쪽지함

수신한 쪽지를 확인합니다. 받은쪽지함 목록은 다음과 같이 구성되어 있습니다.

항 목 설 명
체크박스 쪽지 목록에서 쪽지를 삭제, 보관, 답장 등을 수행할 때 쪽지를 선택합니다.
구분 수신된 쪽지의 읽음/안읽음(/) 상태를 보여줍니다.
보낸이 쪽지를 보낸 사람의 이름 또는 이메일 주소입니다.
내용 쪽지의 내용입니다. 쪽지의 내용을 클릭하면 쪽지 내용이 팝업 창으로 나타납니다.
받은날짜 쪽지의 수신날짜를 보여줍니다.
 

보낸쪽지함

발신한 쪽지를 확인합니다. 보낸쪽지함 목록은 다음과 같이 구성되어 있습니다.

항 목 설 명
체크박스 쪽지 목록에서 쪽지를 삭제, 보관 등을 수행할 때 쪽지를 선택합니다.
구분 발신한 쪽지의 상태를 보여줍니다. 읽음() 상태가 나타납니다.
받는이 쪽지를 받는 사람의 이름 또는 이메일 주소입니다.
내용 쪽지의 내용입니다. 쪽지의 내용을 클릭하면 쪽지 내용이 팝업 창으로 나타납니다.
보낸날짜 쪽지의 발신날짜를 보여줍니다.
수신확인 쪽지를 받은 사람이 쪽지를 읽었는지 여부를 보여줍니다.
쪽지를 읽었을 경우, 쪽지를 확인한 날짜 및 시각을 보여줍니다.
쪽지를 읽지 않았을 경우, 발송취소버튼이 나타납니다. 발송취소버튼을 클릭하면 발송을 취소할 수 있습니다.
 

쪽지보관함

사용자가 보관하는 쪽지를 확인합니다. 쪽지 보관함 목록은 다음과 같이 구성되어 있습니다.

항 목 설 명
체크박스 쪽지 목록에서 쪽지를 삭제, 보관 등을 수행할 때 쪽지를 선택합니다.
구분 송수신된 쪽지의 읽음/안읽음(/) 상태를 보여줍니다.
받는이/
보낸이		 수신 쪽지의 경우 보낸 사람의 이름 또는 이메일 주소를 보여줍니다. 발신 쪽지의 경우 받는 사람의 이름 또는 이메일 주소를 보여줍니다.
내용 쪽지의 내용입니다. 쪽지의 내용을 클릭하면 쪽지 내용이 팝업 창으로 나타납니다.
날짜 쪽지의 수신 또는 발신날짜를 보여줍니다.
 

[참고]
받은쪽지함과 보낸쪽지함에 있는 쪽지는 웹 어드민의 [서비스 > 계정관리 > 그룹관리]와 [서비스 > 계정관리 > 사용자관리]의 쪽지함 유효기간이 지나면 자동으로 삭제됩니다. 쪽지를 장기보관하고자 할 때는 쪽지보관함에 저장하시기 바랍니다.
