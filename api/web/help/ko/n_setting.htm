환경설정

환경설정에서는 쪽지의 차단 정책을 설정하고 수신 허용/거부 사용자를 관리할 수 있습니다.

  1. [쪽지 > 환경설정]메뉴를 클릭합니다.
  2. 쪽지 차단 정책을 설정합니다.
    • 모든 쪽지 허용 - 모든 쪽지를 수신합니다.
    • 모든 쪽지 거부 – 모든 쪽지를 거부합니다.
    • 수신 허용 사용자만 허용 - 수신 허용 목록에 있는 주소에서 온 쪽지만 수신합니다.
    • 수신 거부 사용자만 거부 - 수신 거부 목록에 있는 주소에서 온 쪽지만 거부합니다.
  3. 쪽지 차단 정책을 수신 허용 사용자만 허용 또는 수신 거부 사용자만 거부로 설정한 경우, 수신 허용/거부 설정에서 수신 허용/거부 주소를 등록합니다.
  4. 설정이 완료되면 저장하기버튼을 클릭합니다.
