게시물 읽기

게시판에 등록된 게시물 본문을 확인합니다.

  1. [게시판]메뉴 화면의 왼쪽프레임에서 조회할 게시판을 선택합니다.
  2. 게시물 목록에서 게시물 제목을 클릭합니다.

게시물 본문 구성은 다음과 같습니다.

항 목 설 명
제 목 게시물 제목입니다.
게시물 정보

게시물 기본 정보입니다.

  • 작성자 - 게시물을 작성한 작성자이름
  • 번호 - 해당 게시판에 게시물 등록 번호
  • 조회수 - 게시물 조회 수
  • 작성일 - 게시물을 작성한 일자
첨부파일 게시물 작성 시 추가한 첨부파일입니다.

게시물 전달

게시판에 등록된 게시물을 메일로 발송합니다.

  1. [게시판]메뉴 화면의 왼쪽프레임에서 조회할 게시판을 선택합니다.
  2. 게시물 목록에서 게시물 제목을 클릭합니다.
  3. 게시물 본문의 툴바에서 전달을 클릭합니다.
  4. 해당 게시물 본문을 포함한 메일쓰기 화면으로 전환됩니다.
  5. 제목 입력 및 수신자를 지정합니다.
  6. 툴바에서 보내기 버튼을 클릭하여 메일을 발송합니다.

 
게시물 팝업

게시판에 등록된 게시물을 팝업 창을 통해 내용을 확인할 수 있습니다.

  1. [게시판]메뉴 화면의 왼쪽프레임에서 해당 게시판을 선택합니다.
  2. 게시물 목록에서 게시물 제목을 클릭합니다.
  3. 게시물 본문의 툴바에서 팝업를 클릭합니다.
  4. 팝업 창이 나타나면, 해당 게시물을 확인합니다.

 
게시물 인쇄

게시판에 등록된 게시물을 인쇄합니다.

  1. [게시판]메뉴 화면의 왼쪽프레임에서 해당 게시판을 선택합니다.
  2. 게시물 목록에서 게시물 제목을 클릭합니다.
  3. 게시물 본문의 툴바에서 인쇄를 클릭합니다.
  4. 인쇄 미리보기 창이 나타납니다. 인쇄하기버튼을 클릭합니다.
  5. 인쇄 창에서 인쇄버튼을 클릭합니다.
