1. 메일 읽기

각 메일함의 메일 목록에서 수신 또는 발신된 메일의 본문 내용을 확인합니다.
메일 읽기를 클릭하면 받은메일함으로 이동합니다. 메일함을 클릭하면 해당 메일함에 있는 메일 목록이 나타납니다.
  • 메일함 목록 - 메일함 목록을 클릭하면 개인메일함을 포함한 메일함 목록이 나타납니다. 메일함명을 클릭하면 해당 메일함으로 이동합니다.

  • 전체 선택 - 전체 선택을 클릭하면 해당 페이지에 있는 메일 목록이 모두 선택됩니다. 메일을 전체 선택한 후 삭제, 이동 또는 읽음/안읽음으로 전환이 가능합니다.

  • 읽음/안읽음 - 수신된 메일을 읽음 또는 안읽음으로 표시하여 메일 목록에서 해당 메일을 읽음/안읽음 아이콘으로 구분해줍니다.

  • 답장 - 수신된 메일의 발신자에게 답장 메일을 발송합니다. 답장메일은 수신자의 메일 제목 앞에 Re:로 표시되어 나타납니다.

  • 전체 답장 - 수신된 메일의 발신자 및 참조자 모두에게 답장 메일을 발송합니다. 답장메일은 수신자의 메일 제목 앞에 Re:로 표시되어 나타납니다.

  • 전달 - 수신된 메일을 본문 또는 첨부로 포함하여 다른 사람에게 전달하는 기능입니다. 메일을 전달하면 수신된 메일 제목에 앞에 [Fwd]로 표시되어 나타납니다.

  • 삭제 - 해당 메일함에 있는 메일을 삭제합니다. 삭제된 메일은 휴지통으로 이동됩니다.

  • 이동 - 각각의 메일함에 있는 메일을 받은메일함, 임시보관함, 스팸메일함, 휴지통, 개인메일함으로 이동합니다.

2. 메일 쓰기

메일쓰기 화면에서 수신자 지정, 본문 작성, 그 외 부가 기능을 통해 메일을 작성하여 전송합니다.

  • 받는사람 - 메일 수신자를 지정합니다. 필수 입력 항목입니다.

  • 참조/숨은 참조 - 메일을 참조할 수신 대상 또는 받는사람, 참조자에게 노출되지 않고, 메일을 참조할 필요가 있는 수신자를 지정합니다. 참조/숨은참조 필드는 참조 옆에 (+)를 클릭하면 나타납니다.

  • 나에게 - 로그인한 자신에게 메일을 발송하러면 나에게를 클릭합니다.

  • 제목 - 메일의 제목을 입력합니다.

  • 본문 작성 - 메일 본문을 작성하고, 수신확인, 서명첨부 옵션 등을 설정합니다.
    • 수신확인 - 수신확인을 체크하면 발송한 메일을 수신자가 읽었는지 여부를 확인할 수 있습니다.
    • 서명첨부 - 메일 본문에 서명을 포함하여 메일을 발송하려면, 서명첨부를 체크한 후, 서명을 선택합니다. 웹 메일에 접속하여 서명 등록을 할 수 있습니다. 모바일 기기로 접속할 경우 등록된 서명을 선택할 수만 있습니다.
    • 주소록 - 주소록을 클릭하여 개인주소록 또는 공용주소록에 있는 구성원 개인에게 또는 그룹 전체로 메일을 발송할 수 있습니다.
    • 임시저장 - 작성한 메일을 임시보관함에 저장합니다.
