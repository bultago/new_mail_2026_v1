Other Feature

There is a Basic toolbar above the note list. You are able to delete, save or reply to notes by clicking buttons on the toolbar.

Delete Notes

If you delete notes, it is not possible to recover them. Select notes that you'd like to delete and click Delete on the toolbar.

Store Notes

You can save notes which are in Inbox or Sent. Select notes that you'd like to save and click Store on the toolbar. They will be moved to Stored.

[NOTE]
Notes in Inbox and Sent will be automatically deleted after Valid period for notes of [Services > Account > Group Management] and [Services > Account > Users] in admin page. If you'd like to save notes for a long time, save them in Stored.

[NOTE]
You can save notes as much as The maximum number of storing notes in [Services > Account > Group Management] and [Services > Account > Users].

Reply

You can reply to a note to the sender. Select a note that you'd like to reply to and click Reply on the toolbar. Compose window will be popped up and To field is automatically filled with the sender's email address.

Reject Sender

Add a sender's address to Blocked Addresses. Select a note that you'd like to reject and click Reject Sender on the toolbar.

[NOTE]
Only when Block policy is set as Reject notes from blocking users in [Note > Settings], notes will be rejected if they are from blocked addresses.

Quick Search

You can filter notes easily as read, unread or Sent to myself. Click Quick Search on the toolbar. Condition list including Read, Unread, Sent to myself will be shown. Click one of those conditions.

Unified Search

You can search for notes by entering search term of From, To or Contents. Type search term into the search box and click Search. Search results will be displayed.

[NOTE]
If you click on the search box after searching, the screen will be moved back to the previous step.

Display

You can specify the number of note that will be displayed on a page. Choose the number on the drop-down box above the Basic toolbar.
