Using Filters
Filter Management

 

Filters allows you to manage the flow of incoming messages. Using filters, you can automatically move messages into a certain mailbox.

 
Using Filters

To use filters, follow the steps below.

  1. Click [Options > Filters].
  2. Select whether you will use filters or not.
    • Use, Not Use
  3. Click Save.

 

 
Filter Management

You can add, edit and delete filters.

Add Filters

To create filters, follow the steps below.

  1. Click [Options > Filters].
  2. Enter filter information in Filter Management.

    Item Description
    Sender Address(From),
    Receiver Address(To/CC)    		 Type sender's address or receiver's address.
    (eg) daou_user@daou.co.kr
    Subject Type keywords. Messages containing those keywords will be automatically moved.
    Above messages will be moved to

    Designate a folder where messages will be moved into.

    To add a new folder, follow the steps below:

    1. Check Create a new folder.
    2. Parent Folder - Select parent folder of the folder you created.
    3. Type a new folder name.

  3. Click Save.
  4. A completion message will be prompted. Click OK.


Edit Filters

  1. Click [Options > Filters].
  2. Select a filter you'd like to edit from Filters and click Modify.
  3. Edit filter information in Filter Management.
  4. Click Save.
  5. A completion message will be prompted. Click OK.


Delete Filters

  1. Click [Options > Filters].
  2. Select a filter you'd like to delete from Filters and click Delete.
  3. A confirmation message will be prompted. Click OK.
  4. A completion message will be prompted. Click OK.
