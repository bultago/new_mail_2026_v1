Optional Features

 

You can send an email on Compose Message window after writing a message and setting some options.

 
Compose Message

To write messages, follow the steps below.

  1. Go to [Mail].
  2. Click Compose on the left top of the screen.
  3. When the window changes into Compose window, fill in items below to write a message:
    • Designate Recipient
    • Enter subject
    • Write a message
    • Attach Files
  4. You can set options on the tab menu.
    • Optional features - Read Status, Per Recipient, Save to Sent, Scheduled Delivery, Secure Mail
    • Preview / Save a draft
  5. Click Send on the Write tab menu.

Read Status
If you check Read Status on the Write tab menu, you can check whether recipients read your message or not. If you'd like to see the status, go to [Mail] and click Receipt on the left top of the screen.

Auto save
If you turn on Auto save, your messages will be auto saved in Drafts periodically. You can turn it on and set time interval on the right top of the screen.

  • Turn off, Auto save(30secs), Auto save(1mins), Auto save(3mins)

 
Designate Recipient

You have to designate recipients when sending a message.

  • TO: Recipients of your email message. It is mandatory.
  • Cc: Recipients of your email message.
  • Bcc: Recipients of your email message but they are not displayed to TO and Cc. You can see Bcc if you click next to Cc.

Below describes how to designate recipients.

  • Enter recipients' email addresses in a TO field. (e.g.) test@daou.co.kr
  • Click Addresses to search for email addresses and click names. (e.g.) "Daou Co."<test@daou.co.kr>
  • Select email addresses from Recent.

Ways to Input Recipients

There are three ways to input recipients as below.

Item Description
Automatic Input

When you type some of characters of an email address, the rest of the email address will appear automatically if you've ever sent a message to the same addresses.
Name Input

You can send a message entering recipients' names into TO.
Search Input

You can enter recipients' email addresses after searching for users within the same domain, in Contacts or in Organization.

You can change the way to input recipients in admin page.

  • Sign in as an administrator and go to [Services > Webmail Settings > Default display settings] to change Compose Page.

Below describes the ways in detail.

Automatic Input

When you type some of characters of an email address, the rest of the email address will appear automatically if you've ever sent a message to the same addresses.

Name Input

You can send a message entering recipients' names into TO. The coverage of Name Input is as below.

  • Users within the same domain
  • Contacts, Shared Contacts
  • Recently Sent
  • Organization

[NOTE]
If you do not use Organization, that won't be included for search.

Type recipients' names into TO, Cc, Bcc on a Compose window.
When entering recipients, you have to write them accurately. You can type Korean, English, numeric, two space bars when entering recipients.
 
(e.g.) Suppose recipient's name is Alicia Kim.

  • Alicia Kim(o), Alicia(x), Kim(x)

(e.g.) When there are multiple recipients

  • Alicia Kim, David Cho or Alicia Kim; David Cho

 

[When same names exist]

If recipients have the same name, Recipient window will be popped up with the name and email addresses. Check a name that you'd like to send and click Submit.

[When no contact exist]

If there is no contact by the name you entered, Recipients window will be popped up. If you enter an email address by the name and click Submit, the message will be sent.

 

Search Input

If you type recipients' name or email addresses, they are searched in contacts and organization. The coverage of Search Input is as below.

  • Users within the same domain
  • Contacts, Shared Contacts
  • Recently Sent
  • Organization

[NOTE]
If you do not use Organization, that won't be included for search.

To enter recipients using Search Input, follow the steps below.

  1. Select TO, Cc or Bcc in Recipients on a Compose window.
  2. Enter recipients' name or email address. Enter a name of department if you're using Organization.
    • The length of query term should be between 2 and 128 characters long in Korean and between 3 and 255 characters in English.
    • When searching, results come out matched with what you typed in.
      (e.g.) If you search for a user by the name of Alicia Kim, you should type Alicia Kim. If you type Kim, you'll find no search result.
    • When there are multiple recipients, type comma(,) or semicolon(;) between names.
      (e.g.) Alicia Kim, David Cho or Alicia Kim; David Cho
  3. Click Input or Enter.
    • When same names exist, Recipient window will be popped up with the name and email addresses. Check a name that you'd like to send and click Submit.
  4. An email address you've searched for will be added to Recipients.


TO myself

When you'd like to send an email to yourself, check myself when writing a message.

  1. Check myself next to TO on a Compose window.
  2. Your email address will be filled in TO.

 

Send a Message in Bulk

You can send a message in bulk at once after uploading a TXT file including email addresses.
Only those who have authority to send a message in bulk can send in this way.

[NOTE]
An administrator can assign users who can send an email in bulk in [Services> Domain > Webmail Settings > Mass Mail Sender].

  1. Click Recipient List File Upload at the bottom of a Compose window.
  2. Select a TXT file and upload it.

 
Write a Message

You can write a message body and set items below like attaching signature, V-Card.

Item Description
Editor Mode
  • Select an editor mode between HTML or TEXT.
Signature

If you'd like to attach your signature, check Signature.

[NOTE]
If you create, edit or delete your signature, go to [Options > Signature].
V-Card

If you check V-Card, your V-Card will be attached when sending messages.

[NOTE]
You can manage your V-Card in [Options > General > Compose Message > Attach V-card].
Importance

If you check Importance() when sending a message, there will be a mark appearing before subject.
Character sets

Select an encoding.

  • Korean(EUC-KR), English(US-ASCII), Japanese(ISO-2022-JP), Simplified Chinese(GB2312), Traditional Chinese(BIG5), Unicode(UTF-8)
Featured

Select a feature which will be backgound of the message body.

  • If you click Featured, several letter papers appear.
  • Once you select one, that will appear in the body of the message.
  • If you undo it, click Featured and Cancel.
  • If you'd like not to see the paper, click the paper once more.

[NOTE]
You can manage letter papers in [Domain name > Webmail > Letter] in admin page.

[CAUTION]
If the administrator set Attachment as Apply to default stationery in [Services > Webmail Settings > Stationery] in admin page, three featured images will be attached.
If Max number of attachments in [Setting > SMTP & In/outbound > Default Settings] is below 3 then mails are not sent properly.

[NOTE]
Featured is only available when the editor is set as HTML mode.
Document Templates

You can use document templates which the administrator made and uploaded.

  • If you click Templates, template list will be displayed.
  • Once you click a template name, that will be appear in the body of the message.
  • Click Cancel templates if you'd like to get the template out of the body.
  • Click Close if you'd like to fold the template list.

[NOTE]
Template can be managed in [Services > Webmail Settings > Document Template] in admin page.

[NOTE]
You cannot use Featured and Template simultaneously.

[NOTE]
Document Templates are only available when the editor is set as HTML mode.

 

 
Attach Files

There are two ways to attach files as below.

  • Use ActiveX
    You can use either Open a file or Drag&Drop to upload several files at once. This enables you to attach large files as well.
  • Do not use ActiveX (Text)
    Files are attached by clicking Open a file button. You can attach a file at one time.

Add Files

To upload files to attach, follow the steps below.

  • Click Open a file and select a file from your PC.
  • Click WebFolder and select a file that you'd like to attach.
  • Click Manage large attachment files and add a file. You can see Manage large attachment files when you click Use ActiveX.

    • [NOTE]
    Virus scan for attachment files carry out when attachment file is not large and Activation of Virus Scan in [Filters > Virus] is enabled. As long as the size of attachemt file is below Max File Size to Scan in [Filters > Virus], virus scan can be done.

Delete Files

  • Click a file from the added list and click Delete attachments.

 
Preview/Save a Draft

Preview

You can view a message you are creating before sending.

  1. Click Preview on the Write tab menu.
  2. Preview window will be popped up.

Save a Draft

This provides you with saving your message you're creating in Drafts.

  1. Click Draft on the Write tab menu. Note that subject should be filled to save.
  2. The message is saved in Drafts.

 
Optional Features

 

You can set some options after writing a message.

Read Status

If you check Read Status on the Write tab menu, you can check whether recipients read your message or not.

  • If you'd like to see the status, go to [Mail] and click Receipt on the left top of the screen.

Per Recipient

If the number of recipients is more than one, you can send a message with Per Recipient on.
If you check Per Recipient, recipients only can see his/her email address and others' remain anonymous.

  • Check Per Recipient on the Write tab menu when sending a message.

Save to Sent

When sending a message, the message will be automatically saved in Sent.

  • Check Save to Sent on the Write tab menu when sending a message.

Schedule Delivery

This enables you to designate date and time to send a message.

  1. Click Scheduled Delivery and set date and time.
  2. Click Send.
  3. The message will be sent at the designated time on the designated date.

Secure Mail

You can set password and hint when sending a message. Recipients have to enter the same password to open the message you sent.

[NOTE]
Secure Mail function is not provided by default. You have to purchase an option package for this.

Double Check for Sending

This enables you to double check To, Cc, Bcc and Attachment before sending a message.

[NOTE]
You can activate this function in [Service > Domains > Webmail Settings > Default display settings] in admin page.


  1. Click Send if you finish writing an email. Double Check for Sending window will be popped up.
  2. Check To which is mandatory.
  3. Check Cc.
  4. Check Bcc.
  5. Check Attachment.
  6. Click Send

[Warning]
Please note that if you do not check Cc, Bcc and Attachment even though they exist, then message will be sent without attachment to only To.
