Auto Forward
Auto Forward Settings

 

Auto Forward allows you to forward your incoming messages to other mail addresses.

 
Auto Forward Settings

Set Auto Forward Mode and add forwarding rules

You can add up to 1000 forwarding rules.
To enable or disable Auto Forward, follow the steps below.

  1. Click [Option > Auto Forward].
  2. Select Disable, Leave messages on Server or Delete messages from Server in Forward Mode.
    • Disable - Disable Auto Forward.
    • Leave messages on Server - Save a message and deliver a copy.
    • Delete messages from Server - Do not store a message and directly deliver to other mail addresses.
  3. Add Basic rules. If Forward Mode is set as either Leave messages on Server or Delete messages from Server, mails are automatically forwarded.
    1. Type a mail address into a text box in Basic rules. You can enter either a mail address or a domain.
      eg) daou_user@daou.co.kr, @daou.co.kr
    2. Click Add.
    3. The address will be added into the mail list.
  4. Add Exception rules when you want to forward mails from specific mail addresses or domains.
    1. Select either Senders address (From) or Domain of the senders address (From) and type a mail address or a domain into a text box next to it.
    2. Enter mail addresses which will receive forwarded mails.
      eg) daou_user@daou.co.kr, @daou.co.kr
    3. Click Add.
    4. The addresses will be added into the mail list.

    5. [NOTE]
    You can add forwarding mail addresses as much as Maximum number of auto forwarding address in [Services > Account > Group Management].

  5. Click OK to save settings.

 

Delete forwarding rules

To delete forwarding rules, follow the steps below.

  1. Click [Options > Auto Forward].
  2. Select a mail address from Basic rules. Or select a rule from Exception rules list.
  3. Click Delete.
  4. Click Save.

 

Search Mail Addresses

To search for a mail address from Forward Address List, follow the steps below.

  1. Click [Setting > Auto Forward].
  2. Type text into a text box in Basic rules or Exception rules.
  3. Click Search.
