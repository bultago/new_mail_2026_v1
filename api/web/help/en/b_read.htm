Read Posts

To view a post, follow the steps below.

  1. Click [BBS] on top and select a bulletin board.
  2. Click the subject of a post you'd like to view.

Post consists of items below.

Item Description
Subject The title of the post.
Post Information

Basic information.

  • Author - user's name who created the post.
  • Date - date which created the post.
  • Views - how many times the post has been viewed.
  • Num - sequential number of the bulletin board.
Attachment Files attached to the post.
Body Main content of the post.

 
Forward Posts

This describes how to email posts.

  1. Click [BBS] on top and select a bulletin board.
  2. Click the subject of a post you'd like to email.
  3. Click Forward in Basic tab menu.
  4. Compose window will be popped up including the post.
  5. Select recipients and enter subject.
  6. Click Send in Write tab menu.

 
Pop-up Posts

This describes how to pop-up posts.

  1. Click [BBS] on top and select a bulletin board.
  2. Click the subject of a post you'd like to see.
  3. Click View pop-up in More tab menu.
  4. Read the post once the window is popped up.

 
Print Posts

This describes how to print posts.

  1. Click [BBS] on top and select a bulletin board.
  2. Click the subject of a post you'd like to print.
  3. Click Print in Basic tab menu.
  4. Print window will be popped up. Click Print.
