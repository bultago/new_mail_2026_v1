You can view department hierarchy as well as search members in Organization.

[NOTE]
Organization function is not provided by default. It is available when you purchase an option package for organization.

 
Organization Composition

Organization is composed of items below.

  • Department tree – Department tree is displayed on left of the screen.
  • Members – If a department name is clicked, members in the department will be displayed on right side of the screen.

 
Member Information

Member List

This describes which items are involved in member information.

Item Description
Org Department names where members belong to.
Name

It shows members' names.

  • If you mouse over members' names, their photos will appear.
  • You can view detail information on members if you click members' names.
    • Photo, Name, E-mail, SSN, Mobile, Phone, Department, Title, Class

[NOTE]
Both you and domain administrator can change your personal information. Refer to [Options > Profile] to learn about how to change it.
If you learn about how to change Profile as a domain administrator, refer to User Management from web admin manual.
Title Members' title
Class Members' class
Tel Members' phone number
Mobile Members' mobile number
ID Members' ID. If you click an ID, Compose window will be popped up so that you can email him or her right away.

 

Sorting Members

Members are displayed depending on Sort which is designated by Administrator.

[NOTE]
Refer to Organization in web admin manual to learn about how to sort members.

If you click each item, sort method (descent to ascent or ascent to descent) will be changed.

  • Org, Name, Title, Class, Tel, Mobile, ID

 
Search Member

You can search for a member using member's name, email address, mobile, title or class in Organization.
To search for a member, follow the steps below.

  1. Click [Organization].
  2. Select a search condition.
    • Name, E-mail, Mobile, Title, Class
  3. Enter a query term and click Search.
