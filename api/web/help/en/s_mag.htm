Calendar management allows you to register personal events, meeting schedules and special days. Registered events can be viewed on a daily, weekly or monthly basis so that you can easily add, modify and delete them. Events can be shared with your shared groups.

 
View Calendar

You can view events on a daily, weekly or monthly basis.

Today

You can view today's events.

  1. Click [Calendar] and Today on the top left corner of the screen.
  2. Today's events will be displayed.

Daily

You can view your events on a daily basis.

  1. There are two ways to see your events.
    • Click a date in a small calendar on the left side of the screen.
    • Click Daily tab in [Calendar].
  2. See your events on a time slot basis.
    • If you click an event, details of the event will be displayed.
    • You can simply add an event by click empty area of the calendar.
    • Go to another day by clicking ◀ or ▶ button next to the date on the top of the screen.

Weekly

You can view events on a weekly basis.

  1. Click Weekly tab in [Calendar].
  2. Events will be displayed.
    • If you click an event, details of the event will be displayed.
    • You can simply add an event by click empty area of the calendar.
    • Go to another week by clicking ◀ or ▶ button next to the date on the top of the screen.

Monthly

You can view events on a monthly basis.

  1. Click Monthly tab in [Calendar].
  2. Events will be displayed.
    • If you click an event, details of the event will be displayed.
    • You can simply add an event by click empty area of the calendar.
    • Go to another month by clicking ◀ or ▶ button next to the date on the top of the screen.

 
Add Events

You can create events and set repetition of the events. You can reserve assets when you create events.

  1. There are two ways to create events.
    • Click [Calendar] and Add on the top left corner of the screen.
    • Click empty area of the calendar.
  2. Enter detail information of the event when Add an Event window is prompted.

    Item Description
    Subject Enter the title of an event. It will appear in the calendar.
    Term Enter date and time.
    Setting

    Select options.

    • All day schedule - The event will be continued all day.
    • Repeated schedule - Repetition of the event.
      • Repeated None
      • Daily - The event repeats every day.
      • Weekly - The event repeats every week.
      • Monthly - The event repeats every month.
    • Anniversary - Special days. If you add an event as Anniversary, the subject will appear in red.
    • Reserve assets - If you click Reserve assets, available assets will be displayed. Select an asset you'd like to reserve.
    Location A place where events will occur.
    Contents You can write details of an event.
    Share event The event will be shared with other users.
    • Share this event with other users.– Check if you'd like to share the event.
    • Shared Group - Select a shared group that you'd like to share the event with.
    • Type email address - Add email addresses by typing into to share the event.
    • Notification mails - Check if you'd like to send a mail about the event.

    [NOTE]
    If you share the event with users who are not in the same domain, you can only share the event by sending a notification mail.

  3. Click Add, once you set details of the event button.

 
Modify Events

This is how to modify events.

  1. Click [Calendar] and click an event you'd like to modify.
  2. Event detail window will be popped up.
  3. Click Modify.
  4. Modify an Event will be prompted.

    5. [NOTE]
    For more information about items of the Modify an Event window, please refer to Add Events.

  5. Click Modify, once you set details of the event button.

 
Delete Events

This is how to delete events.

  1. Click [Calendar] and click an event you'd like to delete.
  2. Event detail window will be popped up.
  3. Click Delete.
