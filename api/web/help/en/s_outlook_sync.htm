TMS Calendar Sync enables you to sync events between TMS Calendar and MS Outlook calendar. It offers three types (two-way, one-way) of sync methods.


[NOTE]
TMS Calendar Sync is currently compatible with MS Outlook 2003 and 2007, and operating systems Microsoft Windows 2000 or higher versions.


TMS Calendar Sync Module Download

You must install a TMS Calendar Sync Module provided by Terrace Mail Suite to use TMS Calendar Sync.
Installation instructions is as below.

  1. Click [Calendar > TMS Calendar Sync].
  2. Click Dowload TMS Calendar Sync Module button.
  3. Click the downloaded file when the downloading completes.
  4. Install the module in accordance to the wizard.
  5. Run MS Outlook once the wizard completes.
    Rerun MS Outlook if it is already running.
  6. Once TMS Calendar Sync is installed on your computer, the TMS Calendar Sync Menu will be created on the MS Outlook menu bar.

 
TMS Sync Configuration

Set TMS web mail information and sync configurations on the TMS Sync Configuration menu for syncing. You can also search sync results.

Basic information

Items of Basic Information tab in TMS Sync Configuration are as below.

  1. User Information
    Enter TMS web mail account, password, server IP address.

  2. Running Options
    Turn on/off features you'd like to set.
    3. Item Description
    Auto Sign In

    Turn the Auto Sign In feature on/off. If you turn this feature on, you automatically sign into TMS web mail when MS Outlook starts to run.
    TMS web mail screen

    Turn the TMS web mail screen feature on/off. If you turn this feature on, you should see a TMS web mail screen when MS Outlook starts to run.
    Auto Upgrade

    Turn the Auto Upgrade feature on/off. If you turn this feature on, TMS Calendar Sync Module will be automatically upgraded into your PC.
    Window Auto Close

    Turn the Window Auto Close feature on/off. If you turn on this feature on, sync result window will disappear upon completion of sync.

  3. Sync Mode
    If Manual Sync is set, events are synced only when you click TMS Sync button. If Auto Sync is set, events are synced automatically by periods.

    • Manual Sync – sync events when TMS Calendar Sync button is being clicked.
    • Auto Sync – sync events by periods.
      • Ever 1, 3, 6, 9 hour

  4. Reset Sync History
    Delete all the sync history and sync from the beginning.

Sync Events

Items of Sync Events of TMS Sync Configurations menu are as below.

  1. Sync Options
    Select Sync Method and basic date. You cannot set the end date for syncing.
    2. Item Description
    Sync Method

    There are three types of sync options available between TMS Calendar and MS Outlook calendar.

    • TMS Calendar <-> MS Outlook calendar - All events in your TMS Calendar are added to your MS Outlook calendar. All events in your MS Outlook calendar are added to your TMS Calendar.

    • TMS Calendar -> MS Outlook calendar - All events in your TMS Calendar are added to your MS Outlook calendar.

    • TMS Calendar <- MS Outlook calendar - All events in your MS Outlook calendar are added to your TMS Calendar.
    Basic Date

    Sync events based on basic date.

    • From today - sync events from today.

    • Since TMS Calendar sync Module installed - Sync events after TMS Calendar Sync Module installed.

  2. MS Outlook folder
    Select an MS Outlook folder to be synced.

  3. Sync Options for Conflict
    Conflict happens when you modify a synced event both in TMS Calendar and MS Outlook calendar.

    • Notice of the conflict - A popup window appears to notify you of the conflict and to select which event will be reflected.
    • Reflect TMS Calendar Only - Modified events in TMS Calendar will be reflected in MS Outlook calendar.
    • Reflect MS Outlook Calendar Only - Modified events in MS Outlook calendar will be reflected in TMS Calendar.

  4. Show Sync Result
    Turn the Show Sync Result feature on/off when events are added, deleted and modified.

    • Delete results - Show results when events are deleted.
    • Add/Modify results - Show results when events are added or modified .

Sync Monitor

Items of Sync Monitor tab in TMS Sync Configuration menu are as below.

  1. Lastest Sync Time
    It shows date and time of the latest sync activity.

  2. Search Sync Result

    • Search Period - Select Search Period for searching.
    • Sync Method - Select Sync Method for searching.
      • TMS Calendar <-> Outlook calendar
      • TMS Calendar -> Outlook calendar
      • TMS Calendar <- Outlook calendar
    • MS Outlook folder - Select an MS outlook folder for searching.

  3. Search Result
    It shows search results as long as there are results associated with Search Sync Result.

 
TMS Calendar Sync

Sync events between TMS Calendar and MS Outlook calendar by clicking TMS Calendar Sync button. If [TMS Sync Configuration > Basic Information > Sync Mode] is set as Manual Sync, you could sync event only by clicking this button.

[NOTE]
Events that a TMS web mail user shares are synced but events shared by other accounts are not. Events that shared in MS Outlook are not synced.


[NOTE]
Find detailed information on shared group management and events sharing on Sharing Events.
