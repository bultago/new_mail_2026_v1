You can add tags to your messages. Once messages have tags that you defined, you can view them filtered along tags.

 
Tag Management

You can define tags and can also edit or delete them. There are two ways to manage tags.

  • Manage tags in mailbox list
  • Manage tags in [Options > Folder Management]

    • [NOTE]
    You can create, modify or delete tags in [Options > Folder Management].


Create a Tag

To create a new tag, follow the steps below.

  1. Click Add next to Tags in [Mail].
  2. Tag Setup window will appear.
  3. Enter a tag name and select a color for it.
  4. Click Submit.

Edit a Tag

To edit a tag that you've created, follow the steps below.

  1. EDIT will appear once you mouse over a tag that you'd like to edit in [Mail].
  2. Click EDIT.
  3. Click Modify on a sub menu.
  4. When Tag Setup window appears, enter a new tag name and change a color.
  5. Click Submit when you're done or click Close if you'd like to undo.

Delete a Tag

To delete tags, follow the steps below. Once you delete a tag, messages with the tag do not have it.

  1. EDIT will appear once you mouse over a tag you'd like to delete in [Mail].
  2. Click EDIT.
  3. Click Delete on a sub menu.
  4. A completion message will be prompted. Click OK.

 

 
Add/Remove Tags

Add a Tag

To add tags to your messages, follow the steps below. You have to create tags before adding them.

  1. Go to [Mail] and check messages that you'd like to add tags to.
  2. If you click Tag on the More tab menu, tag list will be popped up.
  3. Select a tag.
  4. The tag you've selected will appear before subject of the messages.

Remove a Tag

To remove tags from your messages, follow the steps below.

  1. Click a tag before subject in [Mail].
  2. Once Delete will be prompted, click it.

 
Filters along Tags

To view messages along tags, follow the steps below.

  1. Go to [Mail] and click a tag that you'd like to see from Tags list.
  2. Messages with it will be displayed.
