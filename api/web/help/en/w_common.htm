Public Web Folder
Public Web Folder Access Authority

 

Public Web Folder is a web folder which all users can access and use.

 
Public Web Folder Access Authority

Public Web Folder can only be edited by some specific users who are granted authority by administrator.

To give authority to specific users, follow the steps below.

  1. Click [Services > Domain List] after you sign in as an administrator.
  2. Click Domain name from the domain list.
  3. Click Webfolder Management.
  4. Add users to User after searching for users.
  5. Click OK.

The users who are granted authority for Public Web Folder can add, edit, delete web folders.

[NOTE]
For more information about how to manage your web folder, refer to [Web Folders > My Web Folder].
