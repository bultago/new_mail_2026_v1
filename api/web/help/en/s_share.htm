Sharing Events

You can share your events with other users. If you share your events with users in the same domain, they can view the events in their calendar.
You can create shared groups before sharing events.

[NOTE]
Please refer to [Shared Group Management] to learn how to create shared groups.

If you want to share your events, follow the steps below.

  1. Click [Calendar] and Add on the top left corner of the screen.
    Or, click a date in a small calendar on the left side of the screen.
  2. Enter details and click Share Event on Add an Event window.
    • The event will be shared.

[NOTE]
Please refer to Figure. Event Details to learn how to create an event.

  1. Select a Shared Group. If you'd like to share the event with users who are not in the same domain, enter an email address to add.
  2. Check Notification mails if you'd like to send a mail about the event.
  3. Click Add once you finish setting an event.

 
Shared Group Management

You can share your events by creating and managing shared groups. Shared groups can be created up to fifty. Shared Group List will be displayed in [Calendar].

You can manage shared groups in menus below.

  • [Calendar > Shared Group]
  • [Options > Calendar]

[NOTE]
Please refer to Calendar to learn how to manage shared groups in [Options > Calendar].

Create a Shared Group

If you want to create a Shared Group, follow the steps below.

  1. Click [Calendar].
  2. Click Add of Shared Group on the left side of the screen.
  3. Add Shared Group window will be prompted.

    Items Description
    Shared Group
    Name

    Enter Shared Group Name.

    • Shared Group name cannot be duplicated.
    • Shared Group name can be 1byte ~ 32bytes in both Korean and English.
    Shared Group
    user

    Designate who you'd like to share your events.

    • Specific user – share your events only with users you choose.
      • Search for a user with ID, Name or Org. If you share events with an organization, events will be shared including sub organizations of the organization that you chose.
      • Add - Choose users from search results and click Add.
      • Delete – Select a user's name from the list you chose and click Delete.

    [NOTE]
    It is available to use Org search as long as you use an organization option.

    • All domain users – share your events with all the users in the same domain.

     
  4. Click Add once you finish setting Add Shared Group.

Modify a Shared Group

If you want to modify shared group information, follow the steps below.

  1. Click [Calendar].
  2. EDIT will appear once you mouse over shared groups' name on the left side of the screen.
  3. If you click EDIT, sub menu will be prompted. Click Modify.
  4. You can modify shared groups' name and shared groups' user on Modify Shared Group window.

[NOTE]
Please refer to Create a Shared Group to learn about items of shared group.

  1. Click Modify once you finish modifying shared group information.

Delete a Shared Group

Once you delete Shared Group, events that belong to the group no longer appear.
If you want to delete shared groups, follow the steps below.

  1. Click [Calendar].
  2. EDIT will appear once you mouse over shared groups' name on the left side of the screen.
  3. If you click EDIT, sub menu will be prompted. Click Delete.
  4. A completion message will appear. Click OK.
