You can create contacts including email addresses and other personal information. You can also share your contacts or view contacts that other users created and shared.
There are two types of contacts as below.

  • Contacts - Your personal contacts.
  • Shared Contacts - Contact multiple users share and use. Users can either manage or view based on access authority.

 
Access Authority

Contacts can be added, modified and deleted based on your access authority level.
There are three authority levels which include Creator, Moderator and Viewer.

Authority Description
Creator
  • Able to create and modify Shared Contacts. Also able to modify and delete Contacts which the Creator added.
  • Able to add, modify and delete groups and members.
  • Able to assign Moderator and Viewer of Shared Contacts.

[NOTE] Able to add Creator in [Service > Domain List > Contacts > Creator Management] in admin page.

Moderator
  • Able to add, modify and delete groups and members of contacts which the Moderator has access authority.
  • Able to assign Moderator and Viewer of contacts which the Moderator has access authority..
Viewer
  • Able to view contacts which the Viewer has access authority.

 

 
Search Contacts

You can search for a member in Contacts and Shared Contacts.

  1. Select a search condition in a text box on the right top of the screen in Contacts.
    • Name, Email, Mobile, Company, Tel
  2. Enter a search keyword and click Search.

 
Send a Message

You can send a message in Contacts.

  1. Go to [Contacts].
  2. Select members or groups from a contact list.
  3. Click Send or Send to all on the Basic tab menu.
  4. Compose window will be prompted.
  5. Click Send once you finish writing a message.

 
Import/Export Contacts

You can easily import contacts by uploading CSV files and also download your contacts from Terrace Mail Suite.

Import Contacts

Upload contacts files from your PC. This functionality is only compatible with MS Outlook Express, Thunderbird or CSV files that MS Outlook supports.

[NOTE]
Please see How to make CSV files_Contacts to learn about contacts format.


To get started, follow the steps below.
  1. Select a group name where you'd like to add contacts to.
  2. Click Import on More tab menu.
  3. Import window will be prompted.
    • CSV - Select a CSV file by clicking Browse.
    • Client - Select a mail client where you saved contacts from. Fields order varies depending on the mail client.
      • TMS, Outlook Express, Outlook
    • Encoding - Select an encoding.
      • euc-kr, shift-jis, utf-8
    • Group - Select a group you'd like to upload to.
  4. Click Browser and select CSV file to upload.
  5. Select a group name.
  6. Click Submit.

Export Contacts

To download contact in TMS, follow the steps below.

  1. Select a group name which you'd like to export.
  2. Click Export on More tab menu.
  3. Click Save.
    • TMS, Outlook Express, Outlook
  4. Encoding - Select an encoding.
    • euc-kr, shift-jis, utf-8

    5. [NOTE]
    Please see Contact Formats by Mail Clients to learn about csv file format when exporting.


  5. Select a folder to download and click Save.

- Import a csv file into MS Outlook
You can export contacts to a comma seperated csv file and import it into MS Outlook. To import a Comma Separated Value (CSV) file into MS Outlook, please follow the steps below.

  1. Export contacts. When exporting, set client as Outlook.
  2. Open Microsoft Outlook.
  3. Select File > Import and Export... menu.
  4. Select Import from another program or file and click Next.
  5. Select Comma Separated Values (Windows) in the list and click Next.
  6. Click the Browse button and locate the CSV (Comma Seperated Values) file you want to import from the location you have it on your computer. Click Open, then click Next.
  7. Click the Contacts folder to import the contacts to it. Click Next.
  8. Click the file that contains your address list and click Map custom fields. Map all the CSV columns into the desired Microsoft Outlook address fields. Drag the column title to the desired field to create new mappings.
  9. Click OK and then click Finish to complete the process.

- Import a csv file into MS Outlook Express
You can export contacts to a comma seperated csv file and import it into MS Outlook Express. To import a Comma Separated Value (CSV) file into MS Outlook Express, please follow the steps below.

  1. Export contacts. When exporting, set client as Outlook Express.
  2. Run MS Outlook Express.
  3. Click File > Import > Address Book menu.
  4. Click on Text File (Comma Separated Values) and then click Import.
  5. Click the Browse button and locate the CSV (Comma Seperated Values) file you want to import from the location you have it on your computer. Click Open, then click Next.
  6. Choose all of the fields that you'd like to import.
  7. Click Finish
