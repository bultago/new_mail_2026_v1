送信メールアドレス管理
送信したメールの受信者メールアドレスを管理します。メールの送信時に使用したメールアドレスが自動で保存され、メール作成画面で“直近のメールを送った宛先”からメールアドレスを選択することができます。

直近のメール送った宛先の削除

送信メールアドレス管理から、メールアドレスを削除します。
  1. [環境設定 > 送信メールアドレス管理]メニューをクリックします。
  2. 直近のメール送った宛先から削除したいメールアドレスを選択します。
  3. 削除ボタンをクリックします。
  4. 削除の完了後、保存ボタンをクリックします。
