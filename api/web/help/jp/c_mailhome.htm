マイメールホーム管理

 

ウェブメールのホームに表示される画面をユーザが希望する画面に設定することができます。

 
マイメールホーム管理

マイメールホームを設定する方法は次の通りです。

  1. [環境設定 > マイメールホーム管理]メニューに移動します。
  2. メールホーム画面にて画面種類を選択します。
    • マイメールホーム – 各画面に表示されるポートレートを選択すると、メールホーム画面に選択したポートレート情報が表示されます。

    参考: マイメールホームのポートレートリストは管理者がウェブ管理サーバにて管理します。ポートレートリスト管理はウェブ管理サーバの [サービス >ドメインリスト > ウェブメール設定 > マイメールホーム管理]で行います。

    • メールホーム – 基本機能メニュー画面を表示します。
    • 受信フォルダ – 受信フォルダにあるメールリストを表示します。
    • フォルダ管理 – フォルダ管理画面が表示されます。 
  3. 選定の完了後、保存ボタンをクリックします。
  4. 保存完了メッセージが表示されます。確認ボタンをクリックします。
