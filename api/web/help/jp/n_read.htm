付箋確認

 

付箋とは同じドメイン内のウェブメールの使用者とやり取りのメモです。

[参考]
付箋機能は製品の購入際、基本的に提供される機能ではありません。別途の購入後、使用できる機能です。

 
付箋確認

送受信付箋の送信(受信)者、送受信日付、付箋内容を確認します。

付箋箱は次のように受信付箋箱、送信付箋箱、付箋保存箱で構成されています。

受信付箋箱

受信した付箋を確認します。受信付箋箱の項目は次のように構成されています。

項目 説明
チェックボックス 付箋リストで付箋を削除、保存、返信などを行う時、付箋を選択します。
分類 受信された付箋の 既読/未読(/)状態を表示します。
送信者 付箋を送信の名前又はメールアドレスです。
内容 付箋の内容です。付箋の内容をクリックすると付箋の内容の画面が表示されます。
受信日時 付箋の受信日時を表示します。
 

送信付箋箱

送信した付箋を確認します。送信付箋箱の項目は次のように構成されています。

項目 説明
チェックボックス 付箋のリストで付箋を削除、保管などを行う時、付箋を選択します。
分類 送信した付箋の状態を表示します。 既読()状態が表示されます。
受信者 付箋を受信者の名前又はメールアドレスです。
内容 付箋の内容です。付箋の内容をクリックすると付箋の内容の画面が表示されます。
送信日時 付箋を送信した日時を表示します。
開封確認 付箋の受信者が付箋を開封したかの可否を表示します。
付箋を開封した場合、付箋を開封した日時及び時間を表示します。
付箋を開封しなかった場合、 送信キャンセルボタンが表示されます。 送信キャンセルボタンをクリックすると送信がキャンセルできます。
 

付箋保存箱

使用者が保存した付箋を確認します。付箋保存箱の項目は次のように構成されています。

項目 説明
チェックボックス 付箋のリストで付箋を削除、保存などを行う時、付箋を選択します。
分類 送受信された付箋の 既読/未読(/)状態を表示します。
送信者/
受信者		 受信付箋の場合、送信者の名前又はメールアドレスを表示します。送信付箋の場合は受信者の名前またはメールアドレスを表示します。
内容 付箋の内容です。付箋の内容をクリックすると付箋の内容の画面が表示されます。
日付 付箋の受信又は送信の日付を表示します。
 

[参考]
受信付箋箱と送信付箋箱にある付箋はウェブ管理コンソール [サービス設定 > アカウント管理 > グループ管理]と [サービス設定 > アカウント管理 > ユーザ管理]の 付箋箱の有効期間が過ぎると自動に削除されます。付箋を長期間保存したい場合は付箋保存箱に保存してください。
