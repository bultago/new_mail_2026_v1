Webメールの主要機能

 
Webメールの主要機能
  • メールの検索機能
    検索対象や検索範囲を指定して検索を行うことができます。添付ファイルの内容まで検索ができます。

  • 下書きフォルダに保存
    現在作成中のメールは送信されずに下書きフォルダに保存されます。 下書き保存されたメールは、下書きフォルダから再編集や送信を行うことができます。

  • 予約送信
    作成したメールを特定の時間に送信する機能です。送信する時間の設定後にメールを送信すると、予約された時間にメールが送信されます。

  • 自動仕分け
    メールの振り分けの条件を設定します。特定条件にマッチしたメールは設定したフォルダ内に保存されます。

  • アドレス帳
    アドレス帳に登録したメールアドレスに簡単にメールを送ることができます。またグループ登録することで、一度に複数のユーザにメールを送ることができます。

  • 共用アドレス帳
    同一ドメインのユーザが使用できる共用アドレス帳を利用できます。

  • メールフォルダ管理機能
    メールフォルダの使用量の確認ができます。メールフォルダを追加し、自動仕分け機能を有効に活用できます。メールフォルダにあるメールは、PCにバックアップできます。バックアップされたメールは、eml形式になっていますので、PCのクライアントでも確認することができます。

  • 開封確認機能
    受信者がメールを開封したかの確認ができます。同一ドメインの受信者に送信したメールは、受信者が開封する前に送信をキャンセルすることもできます。

  • 受信拒否機能
    受信拒否リストに登録した送信者からのメールをスパムメールとして処理されます。

  • 不在通知機能
    不在通知は不在中の使用者が指定した不在期間中の受信メールの送信者に対して、不在通知メッセージを送信します。
