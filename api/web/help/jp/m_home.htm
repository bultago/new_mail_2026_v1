メールホームは、Webメールのログイン後に表示される画面です。メールホームでは、未読メールの件数、メールの使用量、ウェブフォルダの使用量、スケジュール及びお知らせの内容が確認できます。環境設定 > マイメールホームで画面構成を変更することもできます。

画面構成

メールホームの画面構成は次の通りです。

項 目 説 明
受信フォルダ
受信フォルダのメール件数及び未読メールの件数を表示します。
ラストログイン情報
最後にログインした日付、時間及びIPアドレスを表示します。
メールフォルダの容量確認
メールフォルダの容量と使用量を表示します。
ウェブフォルダ
[ウェブフォルダ]の容量と使用量を表示します。
未読メールの件数
メールフォルダ名に、未読メールの件数が表示されます。未読メールの件数をクリックすると、未読メールのリスト画面が表示されます。
スケジューラ
[スケジューラ]で登録した予定の確認ができます。
お知らせ
管理者が登録した、お知らせの内容を確認します。
