グループの管理

アドレス帳のグループを追加、変更、削除することができます。共有アドレス帳の場合、使用権限によってグループを管理する事ができます。

グループの追加

アドレス帳又は、共有アドレス帳にアドレス帳のグループを追加します。

 

[参照] 共有アドレス帳の場合、ウェブ管理サーバで共有アドレス帳を追加しないと該当アドレス帳にグループを追加することができません。共有アドレス帳の管理は [サービス > ドメインリスト]で [該当ドメインクリック > アドレス帳管理 > 共有アドレス帳]で行います。

アドレス帳にグループを追加する方法は次の通りです。

  1. [アドレス帳]メニュー画面、左フレームのグループ横の追加をクリックします。
  2. グループ追加画面にて新しいグループ名を入力します。
  3. 確認ボタンをクリックします。
  4. 追加完了のメッセージが表示されます。確認ボタンをクリックします。

グループ名の変更

アドレス帳のグループ名を変更します。

  1. [アドレス帳] メニュー画面、左フレームのグループ横の管理をクリックします。
  2. グループ管理が表示されます。
    • グループ – 変更するグループを選択します。
    • 新しいグループ名 – 新しいグループ名を入力します。
  3. 確認ボタンをクリックします。
  4. 変更完了のメッセージが表示されます。確認ボタンをクリックします。

グループの削除

アドレス帳のグループを削除します。

  1. [アドレス帳] メニュー画面、左フレームのグループ横の管理をクリックします。
  2. グループ管理画面のグループで削除するグループを選択します。
  3. 削除をクリックします。
  4. 削除の確認メッセージが表示されます。確認ボタンをクリックします。
  5. 削除完了のメッセージが表示されます。確認ボタンをクリックします。

参考: アドレス帳のグループを削除したら、該当グループのみ削除され、グループに属したメンバーはグループ未指定で存在します。

 
グループの管理権限指定

共有アドレス帳のグループを管理及び、参照できる権限を指定します。

参考: グループの使用権限指定は自分が生成したグループや該当グループに管理者の権限が付与されたグループのみが指定が可能です。

共有アドレス帳の管理権限を指定する方法は次の通りです。

  1. [アドレス帳]メニュー画面、左フレームの共有アドレス帳をクリックします。
  2. 共有アドレス帳リストで権限を指定するグループを選択します。
  3. ツールバーの拡張機能タブで参照者又は、管理者をクリックします。
  4. 参照者又は、管理者画面が表示されます。
  5. 権限を付与するメンバーのアカウントを入力した後、追加ボタンをクリックします。
  6. 参照者又は、管理者リストに追加されたら閉じるボタンをクリックします。
