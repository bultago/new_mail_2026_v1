メール検索
基本検索
詳細検索

 

各フォルダにあるメールを検索することができます。

  • 基本検索 – 送信者/受信者、件名でメールを検索します。
  • 詳細検索 – 送信者/受信者、検索期間などの検索条件を設定して検索します。

 
基本検索

基本検索は送信者/受信者、件名の検索キーワードを入力して検索する機能です。

  1. [メール] メニュー画面の右上の検索欄に検索キーワードを入力します。
  2. 検索ボタンをクリックしたら、検索結果リストが表示されます。

[参考]
検索後、検索キーワードの入力欄にあるをクリックすると、検索前の画面に移動します。

 
詳細検索

詳細検索は送信者/受信者、検索期間など検索条件を詳細に検索する機能です。

  1. [メール] メニュー画面の右上の詳細検索をクリックします。
  2. 次のように詳細検索条件画面が表示されます。各検索条件を入力します。

    項目 説明
    送信者 送信者のアカウントを入力します。
    受信者 宛先のアカウントを入力します。CCを含めて検索します。
    検索キーワード

    検索範囲を選択した後、キーワードを入力します。
    フォルダ

    検索するフォルダを選択します。

    [参考] [環境設定 > 基本環境]の検索範囲設定でスパムフォルダ、ゴミ箱を含むにチェックした場合、詳細検索する際フォルダをすべてのフォルダに選択するとスパムフォルダとゴミ箱にあるメールが含まれて検索されます。

    [参考] Web管理画面[環境設定 > その他 > Webメール検索ポリシー]で検索範囲(メールの本文まで、添付ファイルの内容まで)を設定できます。
    検索期間 テキストボックをクリックするとカレンダーが表示されます。日時を選択して検索期間を設定します。
    追加条件 フラグメール、未読メールなどでメールをフィルタリングして検索します。

  3. 検索条件設定の完了後、検索ボタンをクリックします。

[参考]
検索後、検索キーワードの入力欄にあるをクリックすると、検索前の画面に移動します。
