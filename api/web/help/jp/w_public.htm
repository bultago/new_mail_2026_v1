共有ウェブフォルダは他のユーザが共有したウェブフォルダ及び、ファイルを共有権限によって、読み又は、読み/編集を実行できます。

 
共有ウェブフォルダ追加

他のユーザが共有したウェブフォルダを共有ウェブフォルダに追加します。

  1. [ウェブフォルダ]メニューの左フレームでウェブフォルダをクリックします。
  2. ツールバーの拡張機能タブで共有フォルダ参照をクリックします。
  3. 共有フォルダ参照画面が表示されます。
  4. ID又は名前で検索キーワードを入力した後、検索ボタンをクリックします。
  5. 検索結果リストで共有ウェブフォルダに追加する項目を選択します。
  6. 確認ボタンをクリックします。
  7. 追加の完了メッセージが表示されます。確認ボタンをクリックします。

 
共有ウェブフォルダ削除

共有ウェブフォルダに追加したパブリックフォルダを削除します。

  1. [ウェブフォルダ]メニューの左フレームで共有ウェブフォルダに移動します。
  2. 共有ウェブフォルダのツリーで削除するフォルダをクリックします。
  3. ツールバーの拡張機能タブで現在フォルダ解除をクリックします。
  4. 共有フォルダ解除の確認メッセージが表示されます。確認ボタンをクリックします。
  5. 解除完了のメッセージが表示されます。確認ボタンをクリックします。
